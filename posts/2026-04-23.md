## 编辑按语

今天最终成稿的 4 篇论文覆盖了几个彼此独立的方向：dynamic algorithms 里的 algorithmic Lovász Local Lemma，streaming model 下的 decision-tree split computation，chordal graphs 上的 Cluster Vertex Deletion，以及 higher-order tensor / algebra 的 border subrank。

如果勉强找一点局部呼应，前两篇都在问“当输入以时间顺序到来时，维护一个组合对象到底需要多少状态与修复代价”；后两篇则更偏结构理论，一个用 chordal graph 的 clique-tree 与 supermodularity 关掉公开问题，一个把 Strassen 的结构张量视角推到更高阶。除此之外，今天的论文没有单一主题主线。

## Dynamic Construction of the Lovász Local Lemma [arXiv:2604.20836](https://arxiv.org/abs/2604.20836)

这篇论文研究的是 dynamic algorithmic Lovász Local Lemma：固定一个状态空间 $\Omega$，约束集合以 flaw 的形式被对手逐步插入、删除，算法在每次更新后都要输出一个当前无 flaw 的对象。作者证明，只要原来的 static 算法属于那一大类“发现 violation 就局部修复”的 local search / resampling / backtracking 框架，那么把同样的修复过程原样搬到 fully dynamic setting，在 adaptive 甚至 clairvoyant adversary 下仍然收敛得很快。最核心的结论是：总 resampling 次数依然只有近线性级别。更具体地，在论文最一般的 AIS 框架里，总修复步数是 $\tilde O_{\psi,\varepsilon}(q+m+\log |\Omega|)$；在更熟悉的 variable setting 里，则有
$$
\varepsilon s = \tilde O(q+m+M+\log \delta^{-1}),
$$
其中 $q$ 是更新次数，$m$ 是编码所有可能更新序列所需的比特数，$M$ 是由 LLL 参数 $x_i$ 累积出来的 cost。对大多数对称应用，$M=O(q)$，于是总修复步数几乎就是 $q$ 本身。

这个问题的背景要从 LLL 的角色说起。Lovász Local Lemma 最初是一个存在性工具：当一堆坏事件彼此依赖不太强、单个概率也不太大时，虽然直接 union bound 不够用，你仍能断言“有一个对象同时避开所有坏事件”。这类对象在组合优化里到处都是：满足所有 clause 的赋值、合法图着色、没有冲突的调度、没有拥塞的路由。关键是，LLL 关心的往往不是“怎样优化一个数值目标”，而是“怎样在局部约束互相牵扯的情况下，证明全局可行解存在”。

static 世界里，algorithmic LLL 的经典图景已经很成熟。Beck (1991) 首先给出构造性版本；Moser (2009) 和 Moser-Tardos (2010) 把这件事彻底做干净：若状态由独立变量组成，就不断挑一个当前 violated 的 bad event，把它涉及的变量重新随机。这个算法简单得近乎粗暴，但分析极漂亮，因为每次 resample 只动局部，yet 整体不会陷入无穷回滚。随后 Achlioptas, Iliopoulos, and Sinclair (2019) 把这一思路推广成几乎“LLL 风格局部修复算法”的总框架，不再要求只是 product distribution + 变量重采样；Molloy (2019) 则说明 entropy compression 这条线也能被放进类似叙事里，用于 triangle-free graph coloring 这种更精细的着色结果。到这里为止，static 理论的主问题基本已经不是“有没有算法”，而是“哪些局部修复过程也能被同一套 witness/charge machinery 分析”。

dynamic 版本之所以难，不在于“更新后再跑一遍 static 算法”这个想法想不到，而在于 adversary model 立刻变了味。oblivious adversary 预先写死更新序列，分析还接近普通 randomized algorithm；adaptive adversary 则能看到你先前的输出，再决定下一次加哪条约束；clairvoyant adversary 甚至一开始就知道你的整串随机比特。这里一个很自然的误解是：是不是只要保证当前状态看起来仍像“新鲜随机样本”，就能像 static 那样分析？恰好不是。4 月 8 日那篇 [2026-04-08 日报](#2026-04-08) 讨论过 temporal selection attacks，说明在 adversarial dynamics 下维持“fresh-like randomness”本身就可能失败。这篇论文的思路更克制，也因此更强：它不试图证明当前对象分布上接近从头随机采样，而只证明**局部修复链条的总长度**仍然可控。对 LLL 来说，这才是需要的对象。

把模型说得再精确一点：初始时没有 active flaws；第 $t$ 次更新只能 add 或 delete 一个 flaw，于是 active set 从 $F_{t-1}$ 变成 $F_t$；算法必须在更新后输出某个 $\sigma_t\in \Omega\setminus \bigcup_{f\in F_t} f$。删除更新显然不会制造新 violation，所以真正要分析的是插入一个 flaw 后，引发多少后续修补。论文最一般的版本允许 resampling 过程完全脱离“重采样几个变量”这种狭义 MT 语境，只要求它满足 AIS 框架里的 generalized charge 条件。

:::aside[AIS 框架里的 formal 条件]
在 AIS 框架中，fix flaw $F_i$ 从状态 $\sigma$ 跳到 $\tau$ 的转移概率记为 $\rho_i(\sigma,\tau)$。若这一步还额外引入了某个 flaw 集合 $S$，则 generalized charge 定义为
$$
\gamma_i^S
=
\max_{\tau\in\Omega}
\frac{1}{\mu(\tau)}
\sum_{\sigma\in \mathrm{In}_i^S(\tau)}
\mu(\sigma)\rho_i(\sigma,\tau).
$$
这里 $\mu$ 是分析时选取的参考分布。若存在正数 $\psi_i$ 与 slack $\varepsilon>0$ 使得
$$
\frac{1}{\psi_i}\sum_{S\subseteq \Gamma_i}\gamma_i^S\prod_{j\in S}\psi_j \le 1-\varepsilon
$$
对所有 $i$ 都成立，就得到论文的 Condition 1。变量重采样的经典 asymmetric LLL 是它的特例：此时可改写为
$$
\gamma_i \le (1-\varepsilon)x_i\prod_{j\in \Gamma_i\setminus\{i\}}(1-x_j).
$$
:::

论文真正新的地方，是它找到了一种**动态版 witness object**，使 static LLL 分析里最重要的三步还能继续走下去：先把长 execution 压缩成 witness，再算固定 witness 出现的概率，最后对所有 witness 做 union bound。static Moser-Tardos 里的 witness tree 是按 resampling 序列倒序长出来的；那套构造在 dynamic setting 里不再自然，因为每次更新开始时，算法只知道“刚刚新增了一个 flaw”，后面的所有 repair 都应该被归因到这次更新，而不是像 static 那样只围绕某个最终 resample 倒推祖先。作者因此把 witness 改成一个**按更新时间排序的 forest**：第 $i$ 棵树对应第 $i$ 次更新后发生的所有 resamplings；若该次更新是删除，或者新 flaw 并未真的出现，这棵树就是空的；否则根就是这次新加入的 flaw，而后续每个节点的父亲是“引入了它”的那次 resampling。

这个改动看起来不大，但几乎是全文最关键的结构判断。因为一旦根和更新时间绑定，adaptive analysis 的组织方式就彻底变了：算法不再需要证明“某种全局坏历史很少见”，而是把每次更新产生的 repair cascade 单独挂在自己的树根下面，再把这些树拼成 forest。对 variable setting，作者证明：若给定这样一个 proper witness forest，则能唯一定位 resampling table 中哪些随机位被哪些节点消耗；于是该 witness 与随机表相容的概率至多是各节点 flaw 概率的乘积，
$$
\Pr[\tau\text{ is consistent}] \le \prod_{v\in \tau}\Pr[F_v].
$$
接着再用一个 Galton-Watson 式的随机森林生成过程去 upper bound 全部 witness 的总权重，就回到了 Moser-Tardos 分析最熟悉的形状。这给出论文的 Theorem 4.6：若每条可能更新序列都满足 Condition 2，且其 cost 至多为
$$
M=\sum_i \log \frac{1}{1-x_i},
$$
那么
$$
\Pr[\text{总 resamples}>s]\le \delta
\quad\text{whenever}\quad
\varepsilon s=\tilde O(q+m+M+\log \delta^{-1}).
$$

这里最值得停一下的是那个 $m$。它不是数据结构意义上的空间复杂度，而是“所有可能更新序列有多少种”的编码成本。作者先对每个**固定**更新序列证明 resampling 数有指数尾界，再对至多 $2^m$ 条可能序列做 union bound。这个处理非常朴素，但也非常干净；对很多局部更新模型，$m$ 无非是 $O(q)$ 或 $O(qk)$，所以 dynamic 的额外代价并没有重新打开一个大的复杂度缺口。换句话说，这篇论文最深的地方不是发明了某种精巧的 adversary-coupling gadget，而是说明：只要 witness 的根系结构选对了，adaptive dynamics 其实只是 static tail bound 再加一层编码论账本。

对 AIS 的一般框架，事情要微妙得多，因为 resampling 不再是“重抽几个变量”而已。此时单个 flaw 的后效应必须用 point-to-set correlation 来记，也就是 fix 一个 flaw 后可能引入一整个 flaw 集合。作者在这里引入的是 breakage forest：每个节点除了 flaw 标签外，还伴随一个 incoming flaw set $S_v$，表示在那一步 resampling 之前，当前已知但尚未被 collateral fixing 消掉的 flaws。为了让 forest 足以重建 execution，论文要求 flaw selection 规则 `Select` 只能依赖当前 active flaw set 和时间戳，不能依赖 `Address` 刚刚使用的随机位。这个技术条件看着不起眼，实际上是把“execution 可以从 witness 恢复”钉死的地方；没有它，adaptive adversary 与内部随机的耦合会立刻失控。

在这个更一般的 setting 下，作者证明固定 witness forest 的权重可以用 $\gamma_{F_v}^{S_v}$ 的乘积上界，再用一个独立的 witness-generation process 去累加总权重，得到 Theorem 5.1 和 5.2。特别是 adaptive 版本给出
$$
\varepsilon s=\tilde O\!\left(q\log(k\psi)+q\log \psi^{-1}+\log \delta^{-1}\right),
$$
其中 $k$ 是 flaw identifier 的总数上界，$\psi$ 是每次更新后所有 LLL 参数的统一上界。这里能看出这篇论文的框架感：它不只是“把 MT 搬到 dynamic”，而是把整个 AIS machinery 一起搬过去，于是之前那些已经能放进 AIS 或 entropy compression 的 static 算法，原则上都能获得一个 almost black-box 的动态版本。

应用层面，最能体现这一点的是几条 corollary 的跨度。动态 $k$-CNF 直接继承了经典阈值 $d<(1-\varepsilon)(2^k/e-1)$，得到 $\tilde O(dk^2)$ 摊还更新时间。triangle-free 图可以动态维护 $O(\Delta/\ln\Delta)$ list-coloring，更新时间是 $\tilde O(\Delta^3)$；论文也诚实地说明，并行独立工作 Assadi 和 Yazdanyar (2026) 对这个特例做到了更好的专用 bound，但那是一篇问题专用论文，这篇要卖的不是最佳常数，而是“为什么一大块 static coloring/LLL 文献可以整体转成 dynamic”。LMR packet scheduling 那条则更像 showcase：原本 Rothvoss (2013) 的简化证明要串接多层 LLL instance，而本文说明这种“上一层 flawless object 定义下一层 flaws”的复合结构也能动态维护，当 congestion $C$ 和 dilation $D$ 为常数时摊还更新时间甚至是 $\tilde O(1)$。

如果只看 headline result，我觉得最强的一条仍然是 edge coloring。论文证明：若
$$
\varepsilon=\omega\!\left(\frac{\log^{2.5}\Delta}{\sqrt{\Delta}}\right),
$$
则存在 fully dynamic randomized algorithm，在 $q$ 次更新下对最大度 $\Delta$ 的图维护 $(1+\varepsilon)\Delta$-edge coloring，摊还更新时间为
$$
(\Delta(\log q+\log n))^{O(\log(1/\varepsilon))},
$$
而且 adversary 可以是 adaptive。对常数 $\varepsilon$ 且 $\Delta=\mathrm{polylog}(n)$ 的区间，这就是 polylog 级。和论文综述的 prior work 对比，Duan-He-Zhang (2019)、Christiansen (2023)、Bhattacharya-Costa-Panski-Solomon (2024) 这些 near-optimal 动态 edge-coloring 算法都只在 oblivious adversary 下成立；Christiansen (2026) 虽然能处理 adaptive adversary，但在这里关心的 $\Delta=\mathrm{polylog}(n)$ 区间更新时间仍是 $ \exp(\tilde O(\sqrt{\log n}))$。所以本文不是在已有 adaptive 结果上磨一个 log，而是把“adaptive + near-optimal palette + polylog regime”这三件事第一次放到一起。

我读这篇时最强的感受是：它推进的不是某个应用的 best bound，而是“如何分析 dynamic local repair”这件事本身。许多动态算法论文里，真正难的往往是 data structure；这里反过来，算法本身几乎没变，难的是证明这串 repair steps 不会在 adaptive updates 下被 adversary 拉得越来越长。作者的答案不是去维持一种全局 randomness invariant，而是把 repair history 重新编码成以更新时间为根的 witness forest。这个视角一旦接受，为什么 deletion updates 自动免费、为什么 union bound 只多出一个 $m$、为什么同一套分析既包 MT 也包 AIS，就都变得非常顺。

所以我会把这篇论文的价值概括成一句很具体的话：它把“static LLL 分析是一门局部修复的 amortized accounting”这件事，第一次真正提升到了 adaptive fully dynamic setting。以后再看到某个 static resampling / entropy-compression 算法，问题已经不再是“能不能单独为它写一篇 dynamic 版”，而是“它能不能被塞进这篇论文的 witness-forest 账本里”。这比任何单独应用都更像一个新框架真正立住了。

## Nearly Optimal Bounds for Computing Decision Tree Splits in Data Streams [arXiv:2604.20394](https://arxiv.org/abs/2604.20394)

这篇论文研究的是 decision tree 训练里最基础的 primitive：给定一条只允许顺序读一遍的数据流 $(x_1,y_1),\ldots,(x_m,y_m)$，其中 feature 值 $x_i\in[N]$，目标是在所有阈值 $j\in\{0,\ldots,N\}$ 中找一个近似最优 split，把样本分成左边 $\{x_i\le j\}$ 和右边 $\{x_i>j\}$。作者考虑 insertion-only streaming model，不假设数据是 i.i.d.；主结果是：对 regression，能在一遍扫描里用 $\tilde O(M^2/\varepsilon)$ 空间输出 additive-$\varepsilon$ 近似最优 split，并证明这基本是最优的；对 binary classification，Gini impurity 也能做到一遍 $\tilde O(1/\varepsilon)$，并有 matching lower bound；misclassification 的 lower bound 同样是 $\Omega(1/\varepsilon)$，与之前已知 upper bound 合上。
$$
\begin{aligned}
\text{Regression:}&\quad \tilde O(M^2/\varepsilon)\ \text{upper},\ \Omega(M^2/\varepsilon)\ \text{lower};\\
\text{Gini:}&\quad \tilde O(1/\varepsilon)\ \text{upper},\ \Omega(1/\varepsilon)\ \text{lower};\\
\text{Misclassification:}&\quad O(1/\varepsilon)\ \text{upper (prior)},\ \Omega(1/\varepsilon)\ \text{lower}.
\end{aligned}
$$
这里 $\tilde O$ 表示隐藏 polylog 因子。就“一个 streaming primitive 的上界和 matching lower bound 是否都补齐”这个标准看，这篇确实把问题刻画得相当完整。

从应用端说，split selection 是 CART、C4.5、gradient boosting、random forest 这一整系树模型反复调用的最小操作。训练一棵树时，你在每个节点都要做一次“在哪个阈值切最好”；训练很多棵树、很多 feature 并行时，这个操作会被调用成千上万次。在 batch setting 下，把某个 feature 的所有值排好序再扫一遍不难；在 streaming setting 下，样本是持续到来的，内存装不下全部数据，而你又希望每个节点、每个 feature 只占很小的状态。本文讨论的正是这个内存瓶颈，而不是树学习里更常见的 statistical consistency、concept drift 或工程优化。Domingos 和 Hulten (2000) 的 VFDT 开了 stream decision tree 这条线，但它主要研究何时该 split，并且依赖 i.i.d. 流；真正把“单个数值 threshold split 本身要多少 streaming space”作为理论对象来研究的，是 Pham、Ta、Vu (2024) [arXiv:2403.19867](https://arxiv.org/abs/2403.19867) 这篇前作。那篇给了 regression 的 two-pass $\tilde O(M^2/\varepsilon)$ 算法、misclassification 的 one-pass $O(1/\varepsilon)$ 算法，以及 Gini 的 one-pass $O(1/\varepsilon^2)$ 算法。本文的位置很明确：把 regression 从 two-pass 压到 one-pass，把 Gini 从 $1/\varepsilon^2$ 压到 $1/\varepsilon$，同时把三个目标的 one-pass lower bound 都钉住。顺便一提，Tatti (2026) [arXiv:2601.12525](https://arxiv.org/abs/2601.12525) 讨论的是 sparse binary feature 下“如何更快找到近优 split”，那是时间复杂度方向；本文关心的是更基础的 space complexity。

先把对象说具体。decision tree 的一个 threshold split 其实就是“把一维数轴在某个位置切开”。如果是 regression，左边和右边各自只输出一个常数，最好的常数当然就是各自样本均值，因此损失就是左右两边的 within-group variance；如果是 classification，最朴素的损失是 misclassification rate，也就是左右两边各用多数类预测，少数类的那些点算错；Gini impurity 则不是直接数错了多少，而是衡量一个节点“标签混得有多厉害”。节点里若全是同一类，Gini 是 $0$；两类对半开时最脏。对树学习来说，misclassification 更像最终 accuracy，Gini 更像局部纯度，所以 CART 一系算法更常用后者。

:::aside[三种 split loss 的精确定义]
对 split $j$，记
$$
\mu(j)=\frac{\sum_{i=1}^m 1[x_i\le j]\,y_i}{\sum_{i=1}^m 1[x_i\le j]},\qquad
\gamma(j)=\frac{\sum_{i=1}^m 1[x_i>j]\,y_i}{\sum_{i=1}^m 1[x_i>j]}.
$$
回归损失是
$$
L_{\mathrm{reg}}(j)=\frac1m\left(\sum_{x_i\le j}(y_i-\mu(j))^2+\sum_{x_i>j}(y_i-\gamma(j))^2\right).
$$
若是二分类，记左侧正负样本数为 $a,b$，右侧为 $c,d$，则
$$
L_{\mathrm{mis}}(j)=\frac1m\big(\min\{a,b\}+\min\{c,d\}\big),
$$
而
$$
L_{\mathrm{Gini}}(j)=\frac{2ab}{m(a+b)}+\frac{2cd}{m(c+d)}.
$$
空侧按 $0$ 处理。
:::

这类问题里，一个最自然但也最危险的想法是：阈值有 $N+1$ 个，那就维护一堆 summary 去近似每个前缀的统计量。危险在于 regression 和 Gini 都不是只看计数，而是看区间上的一阶、二阶矩；如果真为所有 $j$ 保留足够精细的 prefix 信息，空间很快就回到线性。本文的第一步因此不是“估所有阈值”，而是先证明没必要估所有阈值。核心 Lipschitz 观察非常简单：如果把 split 从 $j$ 挪到 $j'$，中间一共只跨过了 $b$ 个样本点，那么 regression 损失变化至多是
$$
|L(j')-L(j)|\le \frac{bM^2}{m}.
$$
这句话在直觉上很好懂：你只是把 $b$ 个点从右边挪到左边，最多每个点为平方损失贡献 $M^2$ 的量级，所以整体不可能跳得太厉害。于是只要能保证“最优阈值附近每个包含很多点的区间里，都有一个候选阈值落进去”，就已经够了。作者用 reservoir sampling 做的正是这件事：在线维护一个大小
$$
K=\Theta\!\left(\frac{M^2}{\varepsilon}\log N\right)
$$
的均匀样本池，结束后只把采样到的 feature 值及其前一个位置 $x,x-1$ 作为候选。一个标准的“重区间不会被样本完全错过”论证表明，高概率下任何包含超过 $\tau m$ 个点的区间都会被 reservoir 打到；再结合上面的 Lipschitz 性，就能推出候选集里必然存在一个 $j$ 满足
$$
L(j)\le \mathrm{OPT}+\frac{\varepsilon}{16}.
$$
我觉得这一步是全文真正决定成败的地方。它把难点从“如何为全部 $N$ 个前缀保真”改写成“如何抓住少量代表性阈值”，而后者正是 reservoir sampling 最擅长的。

候选集缩小以后，第二个问题才是：怎么在 one pass 里评估这些候选？对 regression，阈值 $j$ 的损失可以写成左右两个区间的 sum squared error 之和，而一个区间的 SSE 只依赖三个量：点数 $n_R$、标签和 $s_R$、平方和 $q_R$。于是作者并行维护三个带 range query 的 dyadic Count-Min sketches，分别对应流 $(x_i,1)$、$(x_i,y_i)$、$(x_i,y_i^2)$。对任一区间 $R$，查询这三张 sketch 得到 $n_R,s_R,q_R$ 的近似值，然后用
$$
\mathrm{SSE}(R)=q_R-\frac{s_R^2}{n_R}
$$
去近似。因为 Count-Min 是单边误差，作者再补一个 $\beta m$ 的校正项，把整个误差压到每个候选 split 不超过 $\varepsilon/4$。于是最终算法的空间就是 reservoir 的 $\tilde O(M^2/\varepsilon)$ 加上三张 sketch 的同量级空间，总体仍是 $\tilde O(M^2/\varepsilon)$。这里最妙的不是用了 Count-Min 本身，而是它和 candidate compression 的分工非常清楚：reservoir 负责把“哪几个阈值值得看”这件事解决掉，sketch 只负责把“看中的这些阈值的区间矩”估出来。两边都很朴素，但配在一起刚好够用。

Gini impurity 部分则有一个非常漂亮的代数塌缩。把标签从 $y_i\in\{-1,+1\}$ 重新编码成
$$
z_i=\frac{y_i+1}{2}\in\{0,1\},
$$
再看同一个 split 的回归平方损失 $L_{\mathrm{MSE}}(j)$，论文直接证明
$$
L_{\mathrm{Gini}}(j)=2L_{\mathrm{MSE}}(j).
$$
这不是渐近等价，而是逐个 split 的精确恒等式。于是 Gini 的最优阈值和 $0/1$ 标签回归的最优阈值完全相同，只差一个常数因子。这样一来，前面 regression 的 one-pass 算法立刻给出 Gini 的 one-pass $\tilde O(1/\varepsilon)$ 上界。和前作相比，这个进步并不是换了一套更细的 Gini 分析，而是发现了一个完全不同的入口：Gini 根本不需要单独做，它本来就是 $M=1$ 回归的另一张脸。这也是我读这篇时最喜欢的一步，因为它让此前 $O(1/\varepsilon^2)$ 的算法显得更像“问题还没被看穿”，而不是某个无法突破的技术障碍。

下界部分同样很整齐。三种损失的 lower bound 都从 communication complexity 里的 Index 出发：Alice 有一个比特串 $z\in\{0,1\}^n$，Bob 有一个位置 $i$，要恢复 $z_i$。作者把这个问题编码成一条数据流，使得流上只有两个相邻阈值 $j^-=2i-1$ 和 $j^+=2i$ 有竞争力，哪一个更优恰好泄露 $z_i$。构造方式也很统一：Alice 在偶数位置 $2k$ 放许多份编码 bit $z_k$ 的“中间块”，Bob 再在 $2i-1$ 和 $2i+1$ 放两个特别大的 anchor blocks，把其他所有 split 都变得明显更差。这样任何 additive-$\varepsilon$ 算法都必须把 $j^-$ 和 $j^+$ 区分开。

三种目标的差别只体现在“为什么这两个阈值之间真的有大于 $\varepsilon$ 的 gap”。对 regression，分析工具是一个简单但关键的恒等式：把一个常值 block 接到某一侧后，SSE 会增加
$$
\frac{aB}{a+B}(\mu-v)^2,
$$
其中 $a,\mu$ 是原 multiset 的大小与均值，$B,v$ 是新 block 的大小与值。这个式子让作者精确算出：当 middle block 落在左边还是右边，平方损失会有 $\Theta(1/n)$ 的差异，于是得到 $\Omega(M^2/\varepsilon)$。对 misclassification，gap 更直接：同一块单色点被放到左侧或右侧，恰好跨过两边的 majority boundary，所以错误数直接差一个 block 大小，得到 $\Omega(1/\varepsilon)$。对 Gini，作者并不重做 gadget，而是用刚才的 re-encoding 恒等式把 Gini lower bound 直接规约到 $M=1$ 的 regression lower bound。整套 lower bound 读下来最舒服的地方，就是它不像很多“matching lower bound”论文那样每个目标都重起炉灶；这里真正的共性结构非常清楚：都是在强迫 streaming algorithm 记住足够多的信息，去分辨一对相邻 split 的细小优劣。

把这些结果放回整个方向里看，本文做成的是一种很标准、也很难得的“问题刻画完成式”推进。前作 Pham、Ta、Vu (2024) [arXiv:2403.19867](https://arxiv.org/abs/2403.19867) 已经把 split selection 从工程 heuristics 抽成了 streaming complexity question；本文把 one-pass regression 与 Gini 的 upper bound 补齐，又把 regression、misclassification、Gini 的 matching lower bound 一起给出。于是对“一维数值 threshold split，在 insertion-only one-pass streaming 里需要多少空间”这个问题，我们现在基本有了完整答案。和 [2026-04-22 日报](#2026-04-22) 那篇 suffix random access paper 一样，这也是那种读完以后你会明确知道“这个 primitive 还剩什么没做”的文章：这里剩下的更像是换目标函数、换 split family、或换 streaming model，而不是在同一问题上再抠一轮主项。就本文本身而言，最值得记住的不是某个 sketch 参数，而是两个极简判断：先用 reservoir 证明“候选阈值其实很少”，再用 $L_{\mathrm{Gini}}=2L_{\mathrm{MSE}}$ 证明“Gini 根本就是 regression”。很多时候，一个问题之所以难，只是因为对象还没被看成它真正该有的样子；这篇论文就属于把对象看对了以后，上下界一起顺下来的一类工作。

## Cluster Vertex Deletion on Chordal Graphs [arXiv:2604.20457](https://arxiv.org/abs/2604.20457)

这篇论文研究的是 Cluster Vertex Deletion (CVD)：给定一个带权图 $G$，删除总权重最小的一批顶点，使剩下的图每个连通分量都是 clique。作者考虑的输入类是 chordal graphs，也就是不含长度至少 $4$ 的 induced cycle 的图；主结果是，CVD 在 chordal graphs 上可以在多项式时间内求解。论文正文里把算法实现成了 $O(n^7)$ 时间，这回答了 Cao et al. (2018)、Aprile et al. (2023)、Chakraborty et al. (2024) 和 Hsieh et al. (2024) 在不同语境下反复追问的那个问题：`split` 和 `interval` 这些较小的 chordal 子类已经会做了，但整个 chordal 类到底是 polynomial-time 还是 NP-hard？

$$
\textbf{Theorem 1.1.}\ \text{Cluster Vertex Deletion on chordal graphs is solvable in polynomial time.}
$$

作者实际求的是一个等价的最大化版本：不是直接问“删谁”，而是问“最多能保留多少权重，使保留下来的诱导子图已经是 cluster graph”。记
$$
\psi(G)=\max\{w(S): S\subseteq V(G),\ G[S]\text{ 是 cluster graph}\},
$$
那么最优删除集就是 $V(G)\setminus S$。这一步改写很自然，因为“保留的部分由若干个互不相邻的 clique 组成”更适合做 dynamic programming。整篇论文的技术也正好分成两层：外层是沿 clique tree 做 DP，内层是一个原本看起来要枚举指数多个顶点子集的子问题，作者证明它其实是 supermodular optimization，于是可以在多项式时间里做完。

先说这个方向为什么存在。所谓 cluster graph，就是每个 connected component 都是 clique 的图；等价地说，它恰好是不含 induced $P_3$ 的图。直觉上它描述的是“数据已经被分成若干个非常干净的团，团内人人相连，团与团之间完全断开”。如果原始图是一个带噪的相似性网络，那么 CVD 就是在问：删掉尽量少的离群点之后，能不能把图清理成若干个真正的团。这也是为什么文献里它一方面被视为 graph-based clustering 的基础模型，另一方面又被算法社区反复拿来当 benchmark：从组合结构上看，它是“打掉所有 induced $P_3$”；从参数化角度看，它定义出的 distance-to-cluster 也是 dense graphs 上少数仍然有信息量的结构参数。

研究脉络里有两条线一直并行。第一条是一般图上的 CVD。Lewis 和 Yannakakis (1980) 早就给出一般 hereditary vertex-deletion 问题的 NP-hardness 框架，因此 CVD 在一般图上当然不可能指望有 polynomial-time 算法。近年的工作更多是 parameterized complexity、kernelization 和 approximation；例如 Aprile et al. (2023) 把近似比从朴素的 $3$ 提到 tight 的 $2$。第二条才是这篇论文所在的位置：把输入限制到特殊图类，看 complexity boundary 究竟落在哪。Cao et al. (2018) [arXiv:1707.08690](https://arxiv.org/abs/1707.08690) 系统研究了 chordal graph 各子类之间的 vertex-deletion 问题，并给出 CVD 在 split graphs 和 interval graphs 上的多项式算法，但 chordal graphs 本身保留为 open。之后 Chakraborty et al. (2024) 又把 split-graph 算法推进到 well-partitioned chordal graphs；这个图类由 Ahn et al. (2022) 引入，目的就是刻画“哪些在 split 上 easy、在 chordal 上可能 hard”的中间地带。另一边，Aprile et al. (2023) 在做 $2$-approx 的过程中把问题压到一些局部更像 chordal 的实例上，并明确提出了 $2P_3$-free chordal 的复杂性问题，甚至猜过一般 chordal 版本也许是 hard 的。本文给出的答案比这还更彻底：不是某个受限 chordal 子类，而是整个 chordal 类都在 P。

| 图类 | 本文之前的状态 | 本文之后 |
| :-- | :-- | :-- |
| split graphs | polynomial-time，Cao et al. (2018) | 无变化 |
| interval graphs | polynomial-time，Cao et al. (2018) | 无变化 |
| well-partitioned chordal graphs | polynomial-time，Chakraborty et al. (2024) | 无变化 |
| chordal graphs | open | polynomial-time，$O(n^7)$ |

这里需要把 chordal graph 本身说清楚。它的定义是“没有长洞”：不存在长度至少 $4$ 的 induced cycle。更重要的是它的结构定理：所有 maximal cliques 可以排成一棵树 $T$，使得每个顶点 $v$ 所在的 maximal cliques 在 $T$ 里连成一个子树。这棵树叫 clique tree。直观上，chordal graph 虽然可以很稠密，但 clique 与 clique 的重叠方式仍然是树状的；这正是很多 chordal-graph 算法的入口。不过 CVD 难做的地方也在这里暴露得很明显：你不是只需要在每个 bag 里维护一个局部模式，而是要维护“最后留下来的图由多少个 clique 组成、它们如何跨越不同 maximal cliques”。如果照 treewidth DP 的惯性去写状态，接口会很快失控。

论文先拿 interval graphs 做了一个很有帮助的 warm-up。interval graph 的 clique tree 退化成一条 path，于是可以谈“最右边的那个 cluster”，再把它从左边已经处理好的部分剥离出来。这个 warm-up 的作用，不是重新证明 interval case，而是帮读者看见一般 chordal case 到底难在哪里：在树上已经没有唯一的“最右边”了，但仍然有一个决定当前子问题值的“那一个 cluster”，只不过它现在可能藏在当前 maximal clique 的整棵子树里。

作者对每个 clique-tree 节点 $K$ 定义了一个不太常规、但非常关键的子问题。把 $T$ 任意定根，设 $P(K)$ 是 $K$ 的父节点，$T_K$ 是以 $K$ 为根的子树，然后定义
$$
G_K = G\!\left[\bigcup_{Q\in T_K}(Q\setminus P(K))\right].
$$
也就是说，子问题只保留“属于 $K$ 这棵子树、但不属于父 clique 的顶点”。这一步一开始看着别扭，其实是整篇 DP 成功的第一处关键设计。因为这样一来，如果 $G_K$ 的最优解根本不碰 $K$，那么它就自动分裂成各个孩子子树上的独立最优解，可以直接求和。作者在文中反复强调，这也是为什么必须把父 clique 里的交界顶点排除掉；不这么做，子问题之间就不会 cleanly decouple。

剩下的情况是最优解确实和 $K$ 相交。由于最终留下来的图是 cluster graph，而 $K$ 本身是 clique，最优解里至多只有一个 cluster 会碰到 $K$。这时问题就变成：把这一个“穿过 $K$ 的 cluster”找出来。论文证明它一定完整地落在某个 maximal clique $Q$ 里，于是可以“猜”这个 $Q$，再从 $K\cap Q$ 里猜一个锚点顶点 $v$，最后决定要不要把 $Q\setminus\{v\}$ 里的哪些点也并进这个 cluster。若记被加入的其余顶点集合为 $X\subseteq Q\setminus\{v\}$，目标函数就是
$$
f_v(X)=w(X\cup\{v\})+\psi\!\left(G_K-N[X\cup\{v\}]\right).
$$
第一项是当前这团 clique 的收益，第二项是删掉它以及它的邻居之后，剩余部分还能取得的最优值。到这里最大的障碍也就完全显形了：$X$ 有指数多个，为什么这个式子不是 brute force？

:::aside[Supermodularity 在这里扮演的角色]
对集合函数 $f:2^U\to\mathbb{R}$，如果对任意 $X,Y\subseteq U$ 都有
$$
f(X\cup Y)+f(X\cap Y)\ge f(X)+f(Y),
$$
就称 $f$ 是 supermodular。把符号取反，$-f$ 就是 submodular function，因此“最大化 supermodular”可转成“最小化 submodular”。本文调用的是 Jiang (2022) 的 oracle-based 算法；在有多项式时间 evaluation oracle 时，可以在多项式时间里完成这一步。
:::

论文真正的技术核心是下面这个结构定理。固定一个 weighted chordal graph $H$ 和其中一个 clique $K$，对每个 $X\subseteq K$ 定义
$$
g(X)=\max\{w(S): H[S]\text{ 是 cluster graph，且 }X\text{ 在 }H[S]\text{ 中恰好构成一个 cluster}\}.
$$
作者证明
$$
\textbf{Theorem 1.2.}\ g\text{ is supermodular on }2^K.
$$
于是对上面的 DP 子问题，只要固定了 $Q$ 和锚点 $v$，函数 $f_v(X)=g(X\cup\{v\})$ 也自动是 supermodular，就不需要枚举 $X$ 了。

我读到这里时最意外的一点是，证明 supermodularity 的方法并不是再做一层 clique-tree bookkeeping，而是一个非常“交换论证”式的全局结构证明。取两个最优解 $S_1,S_2$，分别强制某两个集合 $A_1,A_2\subseteq K$ 成为一个 cluster。然后构造一个二分图 $B$：上侧节点是 $S_1$ 的各个 cluster，下侧节点是 $S_2$ 的各个 cluster；如果原图 $H$ 里两个 cluster 之间存在边，就在 $B$ 里连一条边。关键观察是，$B$ 不能有环；因为一旦在 $B$ 里绕出一个 cycle，就能在 $H$ 里拼出一个长度至少 $4$ 的 induced cycle，和 chordality 矛盾。更准确地说，论文只需要证明连接 $A_1$ 与 $A_2$ 的那条边是 bridge，就已经够用。

一旦这件事成立，后面就很顺了。删掉 $A_1A_2$ 这条 bridge，$B$ 分成两边；作者把两边的 cluster 重新分配，构造出两个新解 $S_\cup$ 和 $S_\cap$，其中一个把 $A_1\cup A_2$ 做成一个 cluster，另一个把 $A_1\cap A_2$ 做成一个 cluster，而且总权重满足
$$
w(S_\cup)+w(S_\cap)=w(S_1)+w(S_2).
$$
于是立刻得到
$$
g(A_1\cup A_2)+g(A_1\cap A_2)\ge g(A_1)+g(A_2).
$$
这一步非常值得细品，因为 chordality 在这里起的不是“给你一个树分解”那种常规作用，而是保证两个全局最优解之间的交互图本身就是树状的。换句话说，真正让 DP 转起来的，不只是 clique tree 的存在，而是 chordal graph 还额外允许你对“候选 cluster 的交换”做出一种离散凸性的控制。

回到算法层面，剩下的事情就很规整了。对每个 $K$，若最优解不碰 $K$，值就是孩子子问题之和；若碰到 $K$，就枚举子树中的 maximal clique $Q$ 和锚点 $v\in K\cap Q$，再对 supermodular 函数 $f_v$ 做一次优化。每次 oracle evaluation 都需要计算 $\psi(G_K-N[X\cup\{v\}])$，这看上去又像个全新难题，但作者用一个简单而关键的引理说明：删掉当前 cluster 及其邻居后，剩余图的每个 connected component 都正好来自 clique tree 里某个更低子树的 $G_Z$，因此可以直接从已经算好的 DP 表里求和。由此总复杂度是 $O(n^7)$：每个 $K$ 有 $O(n)$ 个 $Q$，每个 $(K,Q)$ 有 $O(n)$ 个锚点，Jiang 的算法要做 $O(n^3)$ 次 oracle 调用，而每次调用可在 $O(n)$ 时间内求值。

把这篇工作的贡献放回脉络里看，它做的不是“又把一个 open graph class 填上了”这么简单。以前大家已经知道 CVD 在 split、interval、well-partitioned chordal 上都能做，但这些结果多少还能靠更显式的线性结构或二部覆盖结构推进；到了全 chordal，真正的困难是如何处理一个 cluster 在 clique tree 上的非局部延展。本文给出的答案不是更复杂的状态压缩，而是先把“当前这一个 cluster 的最佳选择”抽象成集合函数，再用 chordality 证明这个集合函数有 supermodular 结构。这个切入点一旦看见，问题就从“怎么在树上记住所有可能的 cluster”变成了“怎么给一个 evaluation oracle”，整个复杂度景观也一下子清楚了。

论文最后顺手提到 $O(n^7)$ 很可能还能继续改进。我也同意这大概不是最终复杂度；但比起常数和次数，真正会留在这个方向里的应该是 Theorem 1.2。它说明在 chordal graph 上，关于“某个指定顶点集能否作为一个 cluster 留下来”的最优值，居然 obeys 一条离散凸性不等式。这个观察和 usual clique-tree DP 的气质很不一样，也正因为这样，它才真的把那个挂了很多年的 open question 关上了。

## Border subrank of higher order tensors and algebras [arXiv:2604.19872](https://arxiv.org/abs/2604.19872)

这篇论文研究的是 higher-order structure tensor 的 border subrank：给定一个有限维代数 $A$，把 $k$ 个元素相乘得到的 multilinear map
$$
T_A^{(k)}:A^{\times k}\to A,\qquad (a_1,\ldots,a_k)\mapsto a_1\cdots a_k
$$
视作一个 $(k+1)$ 阶张量，问它最多能退化出多大的 unit tensor。作者的主结果是把 Strassen 对三阶结构张量的经典工作同时向两个方向推进了：一方面不再只看 asymptotic subrank，而是直接求非渐近的 border subrank $\underline{Q}(T_A^{(k)})$；另一方面把对象从通常的双线性乘法张量 $T_A^{(2)}$ 推广到任意 $k$。对 $k$-fold matrix multiplication，他们证明
$$
\frac{n^2}{k-1}\le \underline{Q}(\mathrm{MaMu}(n,\ldots,n))
\le \frac{1}{2}\bigl(n^2+nq+r(q+1)\bigr),
\qquad q=\left\lfloor\frac{n}{k}\right\rfloor,\ r=n-kq;
$$
对 upper triangular matrix multiplication，他们给出 all $k$ 的精确值
$$
\underline{Q}\!\left(T_{T_n}^{(k)}\right)=\frac{(q+1)(2n-qk)}{2},\qquad q=\left\lfloor\frac{n}{k}\right\rfloor.
$$
此外他们还系统处理了几类交换代数，并证明一个非常关键的 propagation 定理：高阶结构张量之间一旦存在 degeneration，低阶结构张量之间自动也存在。

如果只用一句日常语言来解释这个方向，它研究的是“一个乘法张量里究竟藏了多少条彼此独立的计算通道”。unit tensor $u_{k+1}(q)$ 可以理解成 $q$ 条完全互不干扰的 $k$ 输入乘法线路；所以 $\underline{Q}(T)$ 大，意思不是 $T$ 容易被少量简单张量拼出来，而是它至少能**近似地**模拟出很多条独立线路。这个量和 matrix multiplication complexity 的关系来自 Strassen 的 asymptotic spectra：它既是 laser method 一类技术背后的基本不变量，也是各种 barrier 结果里反复出现的对象。论文引言里还提醒了几条更广的联系：iterated matrix multiplication 是 tensor network 里最自然的 graph tensor；$k[t]/(t^3)$ 的结构张量编码 capset；更高次的 $k[t]/(t^d)$ 与 sunflower 和 multicolored sum-free 问题相关。也就是说，作者研究的不是一个孤立的代数几何玩具，而是一条同时连接 algebraic complexity、combinatorics 和 quantum information 的主线。

:::aside[Border subrank、geometric rank 与 G-stable rank]
若 $u_k(q)=\sum_{i=1}^q e_i^{\otimes k}$ 是 rank-$q$ 的 unit tensor，那么
$$
\underline{Q}(T)=\max\{q: u_k(q)\in \overline{G\cdot T}\}
$$
衡量的是 $T$ 能否退化到多大的 unit tensor。本文反复使用两个上界工具。geometric rank $\mathrm{GR}(T)$ 是某个“零化方程组”解集的余维数；Kopparty, Moshkovitz, and Zuiddam (2020) [arXiv:2002.09472](https://arxiv.org/abs/2002.09472) 证明它上界 ordinary matrix multiplication 的 border subrank。G-stable rank $R_G(T)$ 则来自 invariant theory，可写成一个关于 tensor support 的线性规划，并满足
$$
\underline{Q}(T)\le \mathrm{GR}(T),\qquad \underline{Q}(T)\le R_G(T).
$$
这篇论文的一个重要方法论贡献就是说明：不同代数族里，真正有力的上界工具并不一样。
:::

研究脉络非常清楚。Strassen (1987, 1991) 首次系统引入 subrank / asymptotic subrank，并且在 $k=2$ 时算出了几类结构张量的 asymptotic subrank，包括 ordinary matrix multiplication、upper triangular matrix multiplication、truncated polynomial algebra $ \mathbb{C}[x]/(x^d)$ 和 null algebra。后来 Kopparty, Moshkovitz, and Zuiddam (2020) [arXiv:2002.09472](https://arxiv.org/abs/2002.09472) 用 geometric rank 把**普通** matrix multiplication tensor 的非渐近 border subrank 精确钉到 $3n^2/4$。另一边，Bläser, Mayer, and Shringi (2023) [STACS 2023](https://drops.dagstuhl.de/entities/document/10.4230/LIPIcs.STACS.2023.12) 研究的是 rank 而不是 subrank；他们证明 higher-order structure tensor 的 rank 随 $k$ 的增长要么 polynomial、要么 exponential，取决于 $A/\mathrm{rad}(A)$ 是否可交换。于是本文切中的空白恰好是：higher-order 情形下，subrank / border subrank 这一侧缺的不是某个常数优化，而是一整套对应于 Strassen 与 BMS23 的理论。

论文最核心的组织方式不是“我们又算了几类例子”，而是先给出一个把整篇工作串起来的结构定理。设 $A,B$ 是同维或不同维的 unital associative algebra，如果某个高阶结构张量发生 degeneration，
$$
T_A^{(k_0)}\rightsquigarrow T_B^{(k_0)}\qquad (k_0\ge 2),
$$
那么对所有更低阶 $k\le k_0$ 都有
$$
T_A^{(k)}\rightsquigarrow T_B^{(k)}.
$$
若再加上 $\dim A=\dim B$，那么在任意一个 $k_0\ge 2$ 阶发生 degeneration，事实上会推出**所有** $k$ 阶都发生 degeneration。这件事的重要性在于，它把 higher-order tensor 之间的比较从“逐个阶数分别计算”变成了某种统一的 algebra degeneration 现象。对我来说，这比任何单个数值结果都更说明本文不是局部修补：它在告诉你，higher-order 结构张量之间存在一条此前没有被明确写出来的偏序传播机制。

有了这个 propagation 定理，再看各个族的结果，味道就完全不一样了。对于 local commutative algebra，作者先用 maximal ideal 的 nilpotency 给出一个普适 collapse 现象。若 $A$ 的 socle degree 是 $s$，则当 $k\ge 2s+1$ 时，$\underline{Q}(T_A^{(k)})=1$；在边界情形 $k=2s$ 时也至多是 $2$，而若最高次幂空间是一维，则仍然塌到 $1$。这里的证明非常 invariant-theoretic：不是构造 degeneration，而是证明所有落到 $(\mathbb{C}^{q+1})^{\otimes(k+1)}$ 的 generic restriction 都是 unstable，于是它们不可能退化出 unit tensor。这个思路对 null algebra 和 apolar algebra of a quadric 一下子就给出强结论。对 null algebra $N_n=\mathbb{C}[x_1,\ldots,x_n]/(x_1,\ldots,x_n)^2$，作者得到
$$
\underline{Q}(T_{N_n}^{(k)})=
\begin{cases}
2,& k=2,\ n\ge 2,\\
1,& k\ge 3,
\end{cases}
$$
而 geometric rank 始终是 $2$。对 generalized Coppersmith--Winograd algebra
$$
Q_n=\mathbb{C}[x_1,\ldots,x_n]/(x_ix_j:i\neq j,\ x_i^2-x_1^2:i\neq 1),
$$
作者证明 geometric rank 恒等于 $3$，但 border subrank 却分裂成
$$
\underline{Q}(T_{Q_n}^{(k)})=
\begin{cases}
1,& k\ge 4\ \text{或}\ (k,n)=(3,1),\\
2,& k=3,\ n\ge 2,\ \text{或}\ (k,n)=(2,1),(2,2),\\
3,& k=2,\ n\ge 3.
\end{cases}
$$
这组 case split 很能说明 higher-order border subrank 的新现象：同一个代数族，三阶时还是 Coppersmith--Winograd world 里的重要对象，阶数一升高却会很快塌到 $1$。如果只盯着 geometric rank，你完全看不到这个塌缩。

truncated polynomial algebra $R_d=\mathbb{C}[x]/(x^d)$ 更有意思，因为这里 geometric rank 几乎失灵。作者先把 $T_{R_d}^{(k)}$ 改写成
$$
T_{d,k}=\sum_{i_0+\cdots+i_k=d-1} e_{i_0}\otimes \cdots\otimes e_{i_k},
$$
然后证明
$$
\left\lfloor\frac{d-1}{k}\right\rfloor+1
\le \underline{Q}(T_{R_d}^{(k)})
\le \left\lfloor \frac{2(d-1)}{k+1}\right\rfloor+1.
$$
下界来自一个很漂亮的显式 degeneration：先只保留指数和是 $k$ 的倍数的项，再用以某个中点 $p$ 为中心的 $\varepsilon$-加权，让 AM-GM 和 triangle inequality 同时取等的唯一可能变成 $p_1=\cdots=p_k$，于是只剩下 unit tensor 的对角项。上界则不是 geometric rank，而是 G-stable rank 的一个线性规划解。技术上最能看出作者判断力的地方就在这里：他们没有执着于把 geometric rank 用到底，而是承认这个工具在 $R_d$ 上看不见有用信息，转而换成 G-stable rank。很多论文会把几个不变量并排罗列；这篇不是，它确实解释了“为什么某个不变量在这个族里有效，而另一个不行”。

对 TCS 读者最熟悉的当然还是 matrix multiplication。ordinary matrix multiplication 在 $k=2$ 时已经有 Strassen 下界和 KMZ20 的 tight 上界；本文处理的是 iterated tensor
$$
\mathrm{MaMu}(n,\ldots,n):(X_1,\ldots,X_k)\mapsto X_1\cdots X_k.
$$
他们证明
$$
\frac{n^2}{k-1}\le \underline{Q}(\mathrm{MaMu}(n,\ldots,n))
\le \mathrm{GR}(\mathrm{MaMu}(n,\ldots,n))
=\frac{1}{2}\bigl(n^2+nq+r(q+1)\bigr),
$$
其中 $q=\lfloor n/k\rfloor$，$r=n-kq$。当 $k\mid n$ 时，上界就是 $\frac{k+1}{2k}n^2$。这个 upper bound 的来源很不“matrix multiplication”：作者把零乘积条件 $X_1\cdots X_k=0$ 解释成 equioriented quiver $A_{k+1}$ 上某类表示， irreducible components 与把 $n$ 分成 $k$ 份的 compositions 对应，再证明 balanced composition 使余维数最小。也就是说，ordinary case 里起核心作用的 geometric rank，在 higher-order case 里并不是直接算一个显然的 determinantal variety，而是要经过一层 quiver representation theory 才真正长出 sharp upper bound。与之对照，下界走的是 graph tensor / orthogonal representation 路线：作者给 cycle $C_{k+1}$ 显式构造了一组一般位置正交表示，从而得到 $\Omega(n^2)$ 且带精确 $k$ 依赖的 lower bound。我读到这里时最强的感受是，上下界几乎来自两个完全不同的宇宙，但都恰好抓住了 iterated multiplication 的“链式”结构。

upper triangular matrix multiplication 的结果则是全文里最干净的一段。对 $T_n$ 为 $n\times n$ 上三角矩阵代数，作者得到
$$
\underline{Q}(T_{T_n}^{(k)})=\mathrm{GR}(T_{T_n}^{(k)})
=\frac{(q+1)(2n-qk)}{2},\qquad q=\left\lfloor\frac{n}{k}\right\rfloor.
$$
上界来自一个几乎一眼就能看懂的 blocking：把矩阵分成 $k$ 个对角块，在第 $j$ 个 factor 里把第 $j$ 个对角块清零，乘积自然变成 $0$，于是 geometric rank 立即被压到上述数值。下界更漂亮。作者对基向量 $e_{a,b}$ 施加
$$
e_{a,b}\mapsto \varepsilon^{-(b-a)^2}e_{a,b},\qquad
e_{a,b}^*\mapsto \varepsilon^{k(b-a)^2}e_{a,b}^*,
$$
然后用
$$
k\sum_{t=1}^k (i_t-i_{t-1})^2\ge \left(\sum_{t=1}^k (i_t-i_{t-1})\right)^2
$$
这个 Cauchy--Schwarz 不等式证明：只有所有增量都相等，即
$$
i_1-i_0=\cdots=i_k-i_{k-1},
$$
的项会在极限里存活。于是 surviving summands 恰好由一对参数 $(a,d)$ 标记，数一遍就得到精确值。这里最妙的是，degeneration 最后筛出来的不是一般的 combinatorial pattern，而是“等差路径”。这和 upper triangular matrix multiplication 自身的偏序结构完全贴合，所以公式一旦写出来，就给人一种很难再改动的感觉。

整篇文章还完全求出了 Lie algebra $\mathfrak{sl}_2$ 的 higher-order structure tensor border subrank；不过 `tcs-daily extract` 给出的 `full_text` 在 section 5 附近截断，section 6 的精确公式我无法从正文复核，所以这里不把那部分的 case split 当成已核实内容展开。仅根据 abstract 和 introduction，可以确认作者把 $\mathfrak{sl}_2$ 的所有阶都处理掉了；就论文的整体形状而言，这一节更像是在告诉你：作者的方法并不局限于 associative algebra。

如果要说这篇论文最重要的洞见是什么，我会选“higher-order 之后，border subrank 的世界不再像 $k=2$ 那样由单一上界工具主导”。有些族里 geometric rank 是对的，比如 iterated matrix multiplication 和 triangular multiplication；有些族里 geometric rank 太钝，得换成 G-stable rank；再有些 local algebra，真正决定答案的是 socle degree 带来的 instability collapse。作者没有把这些工具揉成一个抽象黑箱，反而把“哪个工具在哪类代数上有效”这件事本身做成了结论。最后他们还给了一个很能提醒读者不要过度类比 $k=2$ 的例子：对 $n=2,k=3$ 的 iterated matrix multiplication，geometric rank 是 $3$，但 actual border subrank 只有 $2$。也就是说，进入 higher-order 之后，连 ordinary matrix multiplication 这个最经典的例子都开始显露出“upper bound 不再自动 tight”的新地形。本文真正打开的，正是这片地形。

## 未入选但值得关注

### A Quadratic Lower Bound for Noncommutative Circuits [arXiv:2604.20575](https://arxiv.org/abs/2604.20575)
这篇工作对 noncommutative circuits 的 palindrome polynomial 给出二次下界，本身当然值得注意。今天没放进主选，主要是因为从摘要信号看，它更像针对特定对象的单点强化，延展面不如正文里的几篇广。

### Blossom VI: A Practical Minimum Weight Perfect Matching Algorithm [arXiv:2604.20351](https://arxiv.org/abs/2604.20351)
Kolmogorov 的新一代 blossom 实现很可能会对实际求最小权完美匹配产生不小影响。没有收进来，不是因为题目不重要，而是这篇更偏工程与实现表现，今天的版面优先留给能展开讲清楚理论界与技术结构的结果。

### A continuum of Künneth theorems for persistence modules [arXiv:2604.20004](https://arxiv.org/abs/2604.20004)
这篇显然内容很厚，而且带着很强的代数拓扑味道。没选主要因为它离这份日报一贯优先的“明确算法 / 复杂性 headline”稍远，硬塞进今天的主文反而容易写得失焦。

### Designing Approximate Binary Trees for Trees [arXiv:2604.20786](https://arxiv.org/abs/2604.20786)
树上的 approximate binary embedding 模型相当自然，线性时间 4-approx 也是一条干净的结果。只是和今天这几篇相比，它更像一个边界清楚的小推进，而不是需要优先深讲的大型 closing result。
