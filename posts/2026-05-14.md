## 编辑按语

今天的四篇论文覆盖了 learning theory、quantum complexity、graph algorithms 几个不同方向，没有必要硬凑成一个单一主题。Valiant 原始学习模型那篇重新刻画了 positive-only、membership query 和 one-sided error 放在一起时到底什么可学；quantum state isomorphism 那篇则把 orbit decision 问题系统推广到一般群作用，并区分 pure state 与 mixed state 的复杂性。

两篇图算法论文有更直接的技术邻近性：一篇围绕 near minimum cuts 构造 thin spanning tree，另一篇研究单边故障下低成本 directed arborescence 的稀疏 preserver。它们都在处理“全局图结构如何被少量边或少量割证书控制”，但具体问题、工具和目标不同：前者依赖 near-min-cut 的 polygon representation 与 laminar rounding，后者把 replacement shortest paths 和 arborescence 近似拼在一起。

## What is Learnable in Valiant's Theory of the Learnable? [arXiv:2605.13840](https://arxiv.org/abs/2605.13840)

这篇论文研究的是 Valiant (1984) 原始 learning model 中到底哪些 concept class 可学习：给定未知目标概念 $h^\star$，学习器只能从分布 $D$ 中看到正例，允许向 $h^\star$ 做 membership query，最后必须输出一个没有 false positive、但可以漏掉至多 $\varepsilon$ 正例质量的假设。作者的主结果是一个 iff characterization：在有限域，特别是 Valiant 原文的 Boolean hypercube $\{0,1\}^d$ 上，一个概念类序列 $\mathcal H=(\mathcal H_1,\mathcal H_2,\ldots)$ 在这个模型中可学习，当且仅当每个 realizable positive sample 都有 $\mathrm{poly}(d)$ 大小的 adaptive-query compression scheme。这个 characterization 还给出 sample/query complexity 的多项式因子刻画，并说明 membership queries 在这里不仅改善复杂度，而是真的改变了可学习类本身。

形式化地，$\mathcal H$ 在 Valiant model 中可学习，是指存在学习器，对任意 $d,\varepsilon,\delta$、任意 $h^\star\in\mathcal H_d$ 和任意只支撑在 $\mathrm{supp}(h^\star)$ 上的分布 $D$，用
$$
\mathrm{poly}(d,1/\varepsilon,1/\delta)
$$
个正例和 membership queries，输出 $h$，并以至少 $1-\delta$ 的概率满足
$$
\mathrm{supp}(h)\subseteq \mathrm{supp}(h^\star),
\qquad
\Pr_{x\sim D}[x\notin \mathrm{supp}(h)]\le \varepsilon .
$$
第一条是不许把负点误判为正点，第二条是允许保守：可以少覆盖一点目标正区域。这个模型和今天课本里的 PAC learning 很接近，但并不是同一个东西。PAC learner 通常同时看到正例和负例，并允许两边都有 $\varepsilon$ 误差；Valiant 原模型只看正例，却给了主动查询任意点标签的能力，并要求 one-sided error。

为什么这个模型值得重新看？一个原因是历史上的误读很普遍：Valiant 的 1984 CACM 论文常被说成引入了 PAC learning，但它实际定义的机制更接近"positive-only + membership queries + conservative output"。另一个原因是，这三个限制放在一起很自然。很多数据场景中正例容易定义，负例分布却没有 canonical choice；生成模型训练里，我们有大量合法文本，但"非法文本的分布"并不清楚。另一方面，one-sided guarantee 对应一种保守推断：输出的集合必须真包含在目标集合内，不能为了覆盖更多正例而引入 hallucination 式 false positive。Membership query 则代表学习器可以主动测试某些边界点，而不是完全被动地等样本出现。

理解本文结果前，先看三种模型的边界。标准二分类 PAC learnability 由 VC dimension 控制：VC dimension 衡量一个类能在多少点上实现所有 $0/1$ 标记模式，有限 VC dimension 等价于分布无关 PAC learnability；在 $\{0,1\}^d$ 上要求样本复杂度多项式于 $d$，就是要求 $\mathrm{VC}(\mathcal H_d)=\mathrm{poly}(d)$。如果去掉 membership queries，只允许正例，Natarajan (1987)、Shvaytser (1990) 与 Kivinen (1995) 已经刻画了 learnability：关键对象不再是 $\mathcal H$ 本身，而是它的 intersection closure，也就是把 $\mathcal H$ 中任意有限多个概念相交后得到的最小超类。用 Hanneke (2024) 的语言，这等价于 $1$-centered star number $s_1(\mathcal H_d)$ 为 $\mathrm{poly}(d)$。

直觉上，positive-only 且 one-sided 的 learner 最安全的操作是"取所有仍可能为真的目标的交集"。给定正例样本 $S$，所有与 $S$ 一致的假设构成 version space $V(S)$；若没有 query，任何 valid 输出都不能超过
$$
\bigcap_{h\in V(S)} \mathrm{supp}(h),
$$
因为超出去的点可能在某个仍一致的目标里其实是负点。因此交闭包的 VC dimension 控制了这种 closure learner 是否能泛化。本文的核心观察是：membership queries 的作用正是改变 version space。学习器可以通过询问一些点，把 $V(S)$ 切小；如果切完后 version space 的交集已经能"认证"整批正例，那么样本本身就不再携带额外信息。

这个认证被作者定义为 adaptive-query compression。给定 realizable positive sample $S\subseteq\{0,1\}^d$，一个 deterministic adaptive query strategy $\Sigma_S$ 是一棵二叉查询树；每个内部节点问一个点 $x$ 的 membership label，左右分支对应回答 $0/1$。它是 $S$ 的 query compression scheme，如果对每条由某个 $h\in V(S)$ 可实现的响应路径 $r$，都有
$$
S\subseteq \bigcap_{h\in V(\Sigma_S,r)} \mathrm{supp}(h).
$$
换句话说，一旦看到了这条短 transcript，所有仍与 transcript 一致的假设都必须把 $S$ 中每个点标成正。这里"compression"压缩的不是一个 subsample，而是一段交互式证书。它也不是普通 sample compression 的直接替代：普通 compression 可以拿一小撮带标签样本和一些 bits 重建整批标签；但在这里只有正例，若允许任意 reconstruction map，就可以永远输出"全正"，定义会变成空话。所以本文要求 transcript 通过 version space 本身强迫 $S$ 为正。

:::aside[Star number 与交闭包]
对概念类 $\mathcal H\subseteq\{0,1\}^X$，$1$-centered star number $s_1(\mathcal H)$ 是能找到多少点 $x_1,\ldots,x_k$，使得 $\mathcal H$ 中有全正中心概念，并且对每个 $i$ 有一个概念只在 $x_i$ 处从正翻成负、其余这些点仍为正。Hanneke (2024) 证明，包含 $\mathcal H$ 且对有限交封闭的最小可能 VC dimension 正是 $s_1(\mathcal H)$。这让 Natarajan-Shvaytser 的 positive-only characterization 可以写成 $s_1(\mathcal H_d)=\mathrm{poly}(d)$。
:::

主定理可以精确写成：
$$
\textbf{Theorem 1.1.}\quad
\mathcal H=(\mathcal H_1,\mathcal H_2,\ldots)\text{ 在 Valiant model 中可学习}
\iff
\text{每个 realizable }S\subseteq\{0,1\}^d\text{ 有大小 }\mathrm{poly}(d)\text{ 的 adaptive-query compression scheme.}
$$
定量版本也很清楚。若每个 $S$ 有大小 $q(d)$ 的 scheme，则可用
$$
m(d,\varepsilon,\delta)\le q(d)\cdot O\!\left(\frac d\varepsilon \log\frac1\delta\right)
$$
个样本/查询学习。反过来，若已有复杂度 $m(d,\varepsilon,\delta)$ 的 learner，则每个有限 realizable $S$ 有大小
$$
O\!\left(m(d,1/20,1/20)^2 d^2\right)
$$
的 adaptive-query compression scheme。这里的多项式损失来自把普通 learner 强化成能同时对所有潜在目标成功的 learner，再把随机运行编译成 deterministic query tree。

证明中最有意思的一步，是他们没有试图构造某种 one-sided $\varepsilon$-cover。标准 PAC 证明常用 cover 或 ERM，但 one-sided output 需要的是"内逼近"：输出必须包含在目标正区域里。这样的 cover，即对每个 $h$ 找一个 $c$ 满足 $\mathrm{supp}(c)\subseteq\mathrm{supp}(h)$ 且只漏掉 $\varepsilon$ 质量，远不如普通 cover 好理解，而且也看不出 membership query 怎样参与。作者转向 compression 后，问题变成：短交互能否把样本的 positivity 变成 version-space 事实。这一步把 query learning 的味道保留下来，但目标不再是 exact learning 里用 query tree 唯一识别 $h^\star$，而是认证样本已经被所有 surviving hypotheses 接受。

从 learnability 推 compression 的方向分两层。先证 weak query compression：给定 $S$，找到一棵短查询树，使得每条可实现路径至少认证 $S$ 的常数比例。做法是把均匀分布放在 $S$ 上运行一个 weak learner；如果 learner 对这个分布的 false-negative mass 至多 $2/3$，它至少覆盖 $S$ 的 $1/3$。麻烦在于普通 learner 的成功只针对真实目标 $h^\star$，而 compression 要求对路径之后所有仍可能的目标都成立。作者于是证明一个 Valiant-model 版本的 PUAC strengthening：learnable class 可以由一个 learner 学习，使得一次运行以高概率同时对所有与已见 transcript 一致的潜在目标成功。然后固定一条对所有 $h\in V(S)$ 都成功的随机运行，把其中抽到的正例也改写成 membership query，就得到 deterministic weak query strategy。最后递归认证剩余未覆盖点，每轮去掉常数比例，所以深度乘上 $O(\log |S|)\le O(d)$。

从 compression 推 learnability 的方向更像 canonical learner。学习器先取正例样本 $S$，运行对应的 query compression scheme 得到 transcript $T$，再输出 query-closure：
$$
\mathrm{Cl}(T)=\bigcap_{h\in V(T)}\mathrm{supp}(h).
$$
因为真实目标 $h^\star$ 与 transcript 一致，$\mathrm{Cl}(T)\subseteq \mathrm{supp}(h^\star)$，one-sided error 自动成立；剩下只要证明这个输出类能泛化。作者定义 $k$-query closure class
$$
\mathrm{Cl}_k(\mathcal H_d)
=
\left\{
\bigcap_{h\in V(T)}\mathrm{supp}(h):
T\text{ 是长度至多 }k\text{ 的 query transcript},\ V(T)\neq\emptyset
\right\}.
$$
关键引理是
$$
\mathrm{VC}(\mathrm{Cl}_k(\mathcal H_d))\le (d+1)(k+1).
$$
这个 bound 很干净：它完全不依赖 $\mathrm{VC}(\mathcal H_d)$，只来自有限域上长度 $k$ transcript 的计数。这里我觉得最巧的是 closure class 的定义没有把输出看成"至多 $k$ 个 hypothesis 的交"，而是"某段 query transcript 后整个 version space 的交"；前者的 VC dimension 通常会继承原类复杂度，后者却被 transcript 数量控制住。

这也解释了本文所谓 sandwich。PAC learnability 是必要的：若 $\mathrm{VC}(\mathcal H_d)=d^{\omega(1)}$，Valiant model 中也不可学习。Positive-only 无查询 learnability 是充分的：若 intersection closure 的 VC dimension 为 $\mathrm{poly}(d)$，可以完全不用额外查询，从样本里做非交互式认证。但二者都不是等价。作者给出例子证明存在 $\mathrm{VC}(\mathcal H_d)=1$ 却不可学的类，也存在在 Valiant model 中可学但交闭包 VC dimension 为 $d^{\omega(1)}$ 的类。因此 membership query 在这个模型里不是"同一类更快学"的附加资源，而是把可学习类严格扩张到 positive-only 和 PAC 之间。

一般无限域上，论文没有给完整 iff characterization，但给出对应的夹逼定理。若 $\mathcal H\subseteq\{0,1\}^X$ 在 Valiant model 中可学习，则 $\mathrm{VC}(\mathcal H)$ 必须有限，并且定量上
$$
\mathrm{VC}(\mathcal H)\le 9m(1/3,1/3)+1.
$$
若 $X$ 无限，$\mathcal H$ 含全正概念，并且对每个 $x$ 都含有"只把 $x$ 排除、其余点全正"的概念，则不可学习；这给出 VC dimension 只有 $1$ 但仍不可学的反例。另一方面，若交闭包 $\overline{\mathcal H}$ 的 VC dimension 有限，则 $\mathcal H$ 可学习，样本/查询复杂度为
$$
O\!\left(\frac1\varepsilon\left(\mathrm{VC}(\overline{\mathcal H})+\log\frac1\delta\right)\right).
$$
作者还证明这些条件在无限域上都不 tight；这部分更像是在告诉读者：有限域 characterization 的计数骨架不能直接搬到无限域，尤其是 $1$-query closure class 都可能爆出无限 VC dimension。

半空间结果是整篇论文最好的 sanity check。Halfspace 是形如
$$
H^\star=\{x\in\mathbb R^d:\langle n^\star,x\rangle\ge c^\star\}
$$
的线性阈值区域。Natarajan-Shvaytser characterization 已经蕴含：当 $d\ge 2$ 时，$\mathbb R^d$ 上 halfspaces 不能只靠正例在 Valiant one-sided model 中学习。本文证明 membership queries 让它们变得可学习：
$$
\textbf{Theorem 1.6.}\quad
\text{Halfspaces over }\mathbb R^d\text{ can be learned with}
$$
$$
m(\varepsilon,\delta)
=
O\!\left(
\frac{d^2\log d\cdot \log(1/\varepsilon)+\log(1/\delta)}{\varepsilon}
\right),
\qquad
q(\varepsilon,\delta)
=
O\!\left(d^3\log^3 d\cdot \log\frac1{\varepsilon\delta}\right).
$$
下界方面，由于 halfspaces 的 VC dimension 是 $d+1$，Theorem 1.5 给出至少 $\Omega(d)$ 个样本或查询是必要的。算法不是 polynomial time；作者把是否存在 polynomial-time halfspace learner 明确列为 open problem。

几何直觉可以从二维看。拿到正例样本 $S$ 后，取凸包 $C=\mathrm{conv}(S)$。因为所有样本都在未知 halfspace 里，整个 $C$ 也在目标里。若能找到一个简单多边形 $P$，满足
$$
C\subseteq P\subseteq H^\star,
$$
那么输出 $P$ 就没有 false positive；泛化误差则由这类输出集合的 VC dimension 控制。在平面里，对凸包每个顶点 $v$，取 incident 的两条边形成 wedge，再用第三条线把 wedge 截成一个三角形。把一条与目标边界平行的线向凸包滑动，第一次接触会发生在某个顶点；对应 wedge 可以放进 $H^\star$，再选第三条边仍覆盖 $C$。只要用 membership queries 认证三角形顶点为正，就能确认整个三角形在 halfspace 内，因为 halfspace 是凸的。

高维时直接取某个顶点的所有 incident facets 会导致输出复杂度依赖样本数，VC dimension 失控。作者的处理是只选 $d$ 个法向量线性无关的 incident facets，再加一个 closing facet，得到一个 simplex。这样输出族复杂度只依赖 $d$。剩下的困难是候选 simplex 太多，逐个查询会有 $m^{O(d^2)}$ 级别成本；他们把选择正确候选的问题化为有限候选 witness points 的 point-location，用 Kane-Lovett-Moran-Zhang (2020) 一类结果把查询数降到 $O(d\log |X|)$，最终得到上面的 $\mathrm{poly}(d)\mathrm{polylog}(1/\varepsilon\delta)$ 查询界。

还有一个很容易误解、但本文中特别重要的对比：若把 halfspaces 限制在 Boolean cube $\{0,1\}^d$ 上，反而不可学习。原因是 Boolean cube 上的 halfspaces 足够表达"排除某个顶点、接受其他所有顶点"；交这些 halfspaces 就能得到任意子集，于是触发 Corollary 1.4 的不可学习条件。也就是说，$\mathbb R^d$ 情形可学不是因为 halfspaces 更简单，而是因为 learner 能查询样本空间外的几何点，利用凸性拿到证书；在 cube 内部被限制住时，这个几何自由度消失了。

这篇论文给我的主要印象是，它把 Valiant 原模型从一个历史注脚变成了一个有自己 combinatorial object 的模型。Adaptive-query compression 不是把 PAC sample compression 换个名字，而是准确抓住了"正例样本什么时候能被一段交互式证书吸收"。半空间例子则说明这个对象不是纯形式游戏：membership queries 的确可能把一个 positive-only 不可学的经典类变成可学，而且关键不是查询能帮你精确识别目标，而是查询能帮你找到一个足够大的、被认证包含在目标内的保守区域。

## Quantum state isomorphism problems for groups [arXiv:2605.12615](https://arxiv.org/abs/2605.12615)

这篇论文研究的是 quantum state isomorphism：给定两个量子电路制备出的量子态，以及一个有限群 $G$ 在这些态上的 unitary action，判断第一个态能否通过某个群元素变成第二个态。作者把此前只对 symmetric group 研究过的 state isomorphism 推广到一般 group actions，并给出一组复杂性分类：pure-state 版本对任意非平凡有限群都是 $\mathrm{BQP}$-hard，同时落在 $\mathrm{QCMA}\cap\mathrm{QCSZK}$；mixed-state 版本在自然的 faithful representation 条件下是 $\mathrm{QSZK}$-complete；Pauli group 的 pure-state 版本精确为 $\mathrm{BQP}$-complete；Clifford group 版本至少和 Graph Isomorphism 一样难。

形式上，pure-state group isomorphism problem $\mathrm{PSGI}[G]$ 的输入是两个电路 $C_1,C_2$，制备
$$
|\psi_1\rangle=C_1|0^n\rangle,\qquad |\psi_2\rangle=C_2|0^n\rangle,
$$
以及问题定义中固定的一个有效 unitary representation $R:G\to U(2^n)$。给定阈值 $0\le \alpha<\beta\le 1$，要区分：
$$
\exists g\in G:\ \mathrm{Re}\langle\psi_1|R(g)|\psi_2\rangle\ge \beta,
$$
和
$$
\forall g\in G:\ |\langle\psi_1|R(g)|\psi_2\rangle|\le \alpha.
$$
这里 $G$ 不是输入的一部分，而是固定在问题族里的群；输入的 succinctness 来自制备态的量子电路。直觉上，这就是问两个 circuit states 是否处在同一个 group orbit 中。这个定义里保留 $\mathrm{Re}\langle\cdot|\cdot\rangle$ 而不是只看绝对值，是为了不把 global phase 完全 quotient 掉；这点对后面把问题规约到 StateHSP 很关键。

为什么这个问题自然？可以先从经典 isomorphism 问题看。Graph Isomorphism 问两张图是否只差一个顶点重命名；group isomorphism 问两个乘法表是否只差一个元素重命名；这些问题的共同形式是“对象本身的标签不重要，重要的是它在某个对称群作用下的等价类”。在量子信息里也经常遇到同类情形：量子态的物理意义可能只在某组局部基变换、qubit permutation、Pauli/Clifford 变换、或线性光学变换下有意义。于是自然的判定问题不是“两个态是否完全相等”，而是“它们是否在允许的对称操作下相等”。

这条线还连接到 hidden subgroup / hidden shift。普通 Hidden Subgroup Problem（HSP）给一个函数 $f:G\to S$，承诺它在某个子群 $H\le G$ 的 cosets 上常值且不同 cosets 上不同，目标是找 $H$。这是量子算法里解释 Shor 算法和许多 Fourier sampling 算法的统一框架。Hidden Shift Problem 则给两个函数 $f,g$，承诺存在 $s\in G$ 使 $f(x)=g(xs)$，目标是找这个 shift；它比 HSP 更像“两个对象是否由一个 group action 对齐”。本文的 $\mathrm{PSGI}[G]$ 正是 hidden shift 的 state analogue：把函数图表编码成态
$$
|\psi_f\rangle=\frac{1}{\sqrt{|G|}}\sum_{x\in G}|x\rangle|f(x)\rangle,\qquad
|\psi_g\rangle=\frac{1}{\sqrt{|G|}}\sum_{x\in G}|x\rangle|g(x)\rangle,
$$
如果 $f,g$ 只差一个 shift，那么相应的两个态也只差 $G$ 在第一个寄存器上的平移作用。反过来，state isomorphism 允许任意 efficiently preparable states，因此比函数版 hidden shift 更宽。

研究脉络上，Lockhart and González Guillén (2017) 首先研究了 symmetric group 作用下的 quantum state isomorphism [arXiv:1709.09622](https://arxiv.org/abs/1709.09622)：给两个 $n$-qubit states，问是否存在 qubit permutation 把一个变成另一个。他们证明 pure-state permutation isomorphism 是 GI-hard，且在 $\mathrm{QSZK}\cap\mathrm{QCMA}$ 中。这里 $\mathrm{QCMA}$ 是 quantum verifier + classical witness 的 Merlin-Arthur 类；对 isomorphism 问题来说 witness 就是群元素 $g$，verifier 可以制备两个态、作用 $R(g)$、再用 SWAP test 检查 overlap。$\mathrm{QSZK}$ 是 quantum statistical zero knowledge，表示有一个 quantum interactive proof，验证者能确认“非同构”但不学到额外信息。Watrous (2002) 的 Quantum State Distinguishability 是 $\mathrm{QSZK}$ 的 canonical complete problem [arXiv:quant-ph/0202111](https://arxiv.org/abs/quant-ph/0202111)，所以把某个自然问题放进或证明 complete for $\mathrm{QSZK}$，是在给它定位为“量子态可区分性”级别的困难。

另一个近邻是 StateHSP。Bouland, Giurgica-Tiron, and Wright (2024) 引入 quantum state hidden subgroup problem [arXiv:2410.12706](https://arxiv.org/abs/2410.12706)：给定一个态 $|\psi\rangle$ 和群作用，目标是找稳定它的子群 $\mathrm{Stab}(|\psi\rangle)=\{g:R(g)|\psi\rangle=|\psi\rangle\}$。Hinsche, Eisert, and Carrasco (2025) 随后研究 abelian StateHSP 的 generalized Fourier sampling 算法 [arXiv:2505.15770](https://arxiv.org/abs/2505.15770)，并提出 mixed-state 扩展是否仍有高效量子算法的问题。本文的一个直接后果是否定性的：mixed-state StateHSP 在最坏情形下 $\mathrm{QSZK}$-hard，因此除非 $\mathrm{QSZK}=\mathrm{BQP}$，不能期待一般的高效量子算法。

:::aside[几个复杂性类]
$\mathrm{BQP}$ 是多项式时间量子计算。$\mathrm{QCMA}$ 是 classical proof + quantum verification。$\mathrm{QSZK}$ 是允许 quantum messages 的 statistical zero knowledge；$\mathrm{QCSZK}$ 要求通信是 classical messages，但 verifier 仍是 quantum polynomial time。$\mathrm{SZK}$ 是完全 classical 的 statistical zero knowledge。本文一个有意思的点是：pure-state upper bound 从此前 permutation case 的 $\mathrm{QSZK}$ 改成了 $\mathrm{QCSZK}$，也就是把 prover-verifier 通信降成经典信息。
:::

论文的第一组结果可以概括为：
$$
\textbf{Theorem.}\quad
\mathrm{PSGI}[G]\in \mathrm{QCMA}\cap\mathrm{QCSZK}
$$
对任意 efficiently represented finite group $G$ 成立，其中 $\mathrm{QCSZK}$ containment 适用于常数 $\alpha$ 和 $\beta=1-1/\mathrm{poly}(n)$ 的参数区间；而对任意非平凡 $G$，
$$
(\alpha,\beta)\text{-}\mathrm{PSGI}[G]\ \text{is }\mathrm{BQP}\text{-hard}
$$
甚至可取 $\alpha=o(1)$、$\beta=1-\exp(-n)$。这说明 pure-state 版本绝不是 GI 的简单量子包装：只要输入态由任意量子电路 succinctly 给出，哪怕群本身非常小，只要它非平凡，问题已经能编码任意 $\mathrm{BQP}$ 计算。

$\mathrm{QCSZK}$ upper bound 的技术想法很清楚，也比原来的 $\mathrm{QSZK}$ protocol 更有信息。为了证明非同构，verifier 随机选 $j\in\{1,2\}$ 和随机群元素 $g$，制备 $R(g)|\psi_j\rangle$。如果两个态在同一个 orbit 中，那么从随机 orbit element 采样得到的分布完全一样，prover 无法知道 $j$；如果两个 orbit 彼此远，unbounded prover 可以识别它来自哪个 orbit。Lockhart-González Guillén 的版本是把量子态副本发给 prover。本文改为发送 classical shadows：verifier 对 $R(g)|\psi_j\rangle$ 做随机 Clifford 测量，把经典测量结果发给 prover。Huang, Kueng, and Preskill (2020) 的 classical shadow 框架保证，用 $O(\log |G|)$ 个 shadow 可以同时估计它与所有 orbit candidates 的 fidelity。这样 prover 仍能在非同构时识别 $j$，而通信已经全是 classical。

这里的关键限制是 pure state。对 mixed states，classical shadow 估计 fidelity 的样本复杂度会受 effective rank 影响，不能直接给出同样的 classical-message protocol。这个分叉正好预示了后面的 $\mathrm{QSZK}$-completeness：mixed-state 版本不是“同一个问题加一点噪声”，而是真的落到 Quantum State Distinguishability 的难度层级。

Pure-state lower bound 的核心是把 BQP computation 嵌入“是否在 orbit 附近”。给定一个 BQP circuit $Q$，放大后可以让它在 yes/no 两种情形下把 $|0^n\rangle$ 分别映到接近 $|0^n\rangle$ 或某个态 $|\phi\rangle$。只要选 $|\phi\rangle$ 使它同时远离所有 $R(g)|0^n\rangle$，那么比较 $Q|0^n\rangle$ 与 $|0^n\rangle$ 就得到一个 state isomorphism instance。对 Pauli 或 Clifford 这类固定群，可以用 magic state 作为 $|\phi\rangle$；对一般 finite group，作者用 approximate state design 随机选一个态，并用 union bound 说明它高概率远离整个 orbit。因此一般 theorem 是 randomized reduction。这个证明让我觉得重要的点是：hardness 并不是来自群结构复杂，而是来自 circuit-state 输入模型本身的表达力；群只需要提供一个非平凡 action，让“接近某个 orbit”成为一个可区分的 promise。

Abelian group 情形下，本文把 $\mathrm{PSGI}[G]$ 和 StateHSP 接上。若 $G$ abelian，给两个态 $|\psi_1\rangle,|\psi_2\rangle$，构造
$$
|\Psi\rangle=\frac{1}{\sqrt 2}\bigl(|0\rangle|\psi_1\rangle+|1\rangle|\psi_2\rangle\bigr)
$$
并让 generalized dihedral group $G\rtimes \mathbb Z_2$ 作用在这个额外 label qubit 和原系统上。直观上，$\mathbb Z_2$ 的非平凡元素负责交换两条 branch，$G$ 的元素负责在交换时补上 shift。于是存在 $h\in G$ 使 $R(h)|\psi_2\rangle=|\psi_1\rangle$，当且仅当 stabilizer 里存在一个 “odd” element $(h,1)$。这就是 hidden shift 到 HSP 的经典规约在 state setting 下的翻译。robust 版本取 $m$ 份 tensor power，把 yes case 的 overlap $(1-\varepsilon)$ 变成至少 $(1-\varepsilon)^m\ge 1-m\varepsilon$，把 no case 的 soundness 压成 $\alpha^m$。

这个 reduction 最具体的 payoff 是 Pauli group：
$$
\textbf{Theorem.}\quad
\text{若 }\alpha<1\text{ 为常数且 }\beta=1-o(1/(n\log^2 n)),\text{ 则 }
(\alpha,\beta)\text{-}\mathrm{PSGI}[\mathcal P_n]\text{ is }\mathrm{BQP}\text{-complete}.
$$
Pauli group 本身不是 abelian，因为两个 Pauli 可能反交换；但两份拷贝 $\mathcal P_n^2=\{P\otimes P:P\in\mathcal P_n\}$ 是 abelian，因为符号在 tensor square 里抵消。作者先把态换成两份拷贝，再套上面的 StateHSP reduction。最终得到的群基本是 $\mathbb Z_2^{2n+2}$，Fourier sampling 只是 Hadamard transform 加线性代数。这里最巧的是“两份拷贝 abelianize Pauli”这一步：它不是把非交换性近似忽略掉，而是精确地把 commutator phase 消掉，让 hidden-shift machinery 可以无损接入。

Clifford group 走向完全不同。作者证明 $\mathrm{PSGI}[\mathcal C_n]$ 是 GI-hard。直接用 graph states 不行，因为所有 stabilizer states 之间在 Clifford action 下关系太多；即使两张图不 isomorphic，相应 graph states 也可能被某个 Clifford 关联。论文的修正是把 graph state 和 magic-state anchor 叠在一起：
$$
|\psi_i\rangle=\frac{1}{\sqrt 2}\bigl(|R\rangle|R^n\rangle+|R^-\rangle|G_i\rangle\bigr),
$$
其中 $|G_i\rangle$ 是图 $G_i$ 的 graph state，$|R\rangle=(|0\rangle+e^{\pi i/8}|1\rangle)/\sqrt 2$。若 $G_1,G_2$ isomorphic，顶点 permutation 显然把一个态变成另一个。反方向要证明：任何高 overlap 的 Clifford 都必须近似保持 $|R^n\rangle$，而能近似保持这种 magic product state 的 Clifford 只能像 permutation 那样作用；这样它在 graph-state branch 上也只能实现图同构。这个 gadget 的功能很具体：magic-state branch 把 Clifford group “钉回” permutation subgroup，graph-state branch 再承载 GI hardness。

Mixed-state 版本的定义把 pure-state overlap 换成 fidelity。输入是两个 mixed states $\rho_1,\rho_2$ 的制备电路，判断是否存在 $g$ 使
$$
F(\rho_1,R(g)\rho_2R(g)^\dagger)\ge \beta
$$
或对所有 $g$ 都至多 $\alpha$。主结果是：
$$
\textbf{Theorem.}\quad
\mathrm{MSGI}[G]\text{ is }\mathrm{QSZK}\text{-complete}
$$
对任意非平凡 efficiently representable finite group 的 faithful projective action 成立；论文中更精确地用
$$
\mu=\max_{g\ne I}\frac{|\mathrm{Tr}(R(g))|}{2^n}<1
$$
排除“某个非单位群元素只作用成 global phase”这种在 mixed states 上完全不可见的退化，并给出 $\alpha>1/2+\mu/2$、$\beta>\alpha$ 的参数条件。

Containment in $\mathrm{QSZK}$ 仍然是 orbit randomization protocol：verifier 随机选择 $\rho_1$ 或 $\rho_2$，随机作用 $g$，发送若干份量子态给 prover。如果两者 isomorphic，twirled ensembles 完全相同；如果所有 group shifts 下都 fidelity 小，那么 $k$ 份 tensor power 之后两个 twirled states 的 fidelity 被压到很小。论文用 Rotfel'd trace inequality 得到一个类似 fidelity union bound 的估计，粗略地说
$$
F\left(\rho_1,\frac{1}{|G|}\sum_{g\in G}R(g)\rho_2R(g)^\dagger\right)
\le \alpha |G|,
$$
再通过 tensor power 取 $k=\omega(\log |G|)$ 放大区分度。

Hardness 则从 Quantum State Distinguishability 规约。给 QSD instance $\sigma_1,\sigma_2$，构造
$$
\rho_1=\frac12\sigma_1+\frac12\psi,\qquad
\rho_2=\frac12\sigma_2+\frac12\psi,
$$
其中辅助态 $\psi$ 被选得远离任何非平凡 group action 的 invariant behavior。若 $\sigma_1,\sigma_2$ 接近，identity 已经给出 isomorphism；若它们远，则任何非 identity group element 都会破坏 $\psi$ branch，而 identity 又无法把 $\sigma_1$ 变成 $\sigma_2$。这类“加一个 anchor branch 强迫 isomorphism 只能取特定形式”的设计，和 Clifford/GI reduction 里的 magic anchor 是同一种证明习惯，只是这里 anchor 用来杀掉所有非 identity actions。

:::aside[为什么 mixed-state StateHSP 也变难]
对含 involution 的 abelian group，作者把 QSD instance $\sigma_1,\sigma_2$ 编码进
$$
\rho=\frac12|v_1\rangle\langle v_1|\otimes\sigma_1+
\frac12|v_2\rangle\langle v_2|\otimes\sigma_2,
$$
其中某个二阶元素 $h$ 交换 label states $|v_1\rangle,|v_2\rangle$。若 $\sigma_1,\sigma_2$ 接近，则 $h$ 近似稳定 $\rho$；若它们远，则不稳定。于是能解 mixed StateHSP 就能解 QSD，得到 $\mathrm{QSZK}$-hardness。
:::

论文最后还开了一个 infinite-dimensional 版本：bosonic linear optical isomorphism。Bosonic states 生活在 Fock space，基向量记录每个 mode 里有多少 photons。对有限 photon number 的 core states，可以用 stellar representation 里的 polynomial 表示；passive linear optical unitary 对应 $U(n)$ 对变量 $z$ 的线性替换。问题变成：给两个 $r$-photon core state polynomials，是否存在 $U\in U(n)$ 使它们相差这个变量替换。作者证明 $r=2$ 已知可解，而 $r=3$ 已经 GI-hard；一般 $r$ 在合适 promise 下落在 $\mathrm{NP}\cap\mathrm{SZK}$。

GI-hardness 的构造也很有辨识度。给图的邻接矩阵 $A^{(1)},A^{(2)}$，构造三光子态
$$
|\psi_i\rangle\propto \sum_k |3_k\rangle+\sum_{p,q}A^{(i)}_{pq}|1_p1_q\rangle.
$$
如果只用二次项 $\sum A_{pq}|1_p1_q\rangle$，线性变换能看到的更接近谱信息，不足以刻画 GI；加入 $\sum_k|3_k\rangle$ 后，任何保持这部分三次结构的线性变换必须接近 permutation up to phases。这用到了 $p\ne 1,2$ 时 $\ell_p$ norm 的线性等距基本上只有置换和相位这一事实。于是三光子项承担了和前面 magic-state anchor 类似的角色：把连续的 $U(n)$ action 收缩到离散的 permutation action，再让二次项表达图邻接关系。

我读这篇论文时最有用的理解方式，是把它看成一张 orbit-decision 问题的复杂性地图。Pure states 里，classical shadow 让非同构证明可以只传经典消息；mixed states 里，同样的 orbit randomization 却直接撞上 $\mathrm{QSZK}$-complete 的 Quantum State Distinguishability。Pauli group 由于 tensor-square abelianization 精确落到 $\mathrm{BQP}$；Clifford group 因为包含足够多 permutation-like structure，至少承载 GI；linear optics 则显示连续群也会在低 photon number 处出现同样的离散 isomorphism 难度。这个分类最扎实的地方不是某一个 bound 特别意外，而是同一个“给 orbit 加 anchor、再看 twirling 后可区分性”的证明语言在有限群、mixed states 和 bosonic states 三个场景里反复出现。

## Thin Trees for Near Minimum Cuts [arXiv:2605.12669](https://arxiv.org/abs/2605.12669)

这篇论文研究的是 thin spanning tree 问题的一个受限版本：给定一张 $k$-edge-connected 图 $G=(V,E)$，目标是找一棵生成树 $T$，使得 $T$ 在每个“接近最小”的割上只用常数条边。作者证明，当 $\eta=1/40$ 时，可以在多项式时间内构造一棵生成树 $T$，满足对所有 $\eta$-near minimum cut $S$，也就是所有 $|\delta(S)| < (1+\eta)k$ 的割，
$$
|T\cap \delta(S)| \le 88.
$$
由于这些割本身大小至少是 $k$，这等价于在 near minimum cuts 这一族割上得到 $O(1/k)$-thinness。它不是完整的 strong thin tree conjecture，但它在最紧的割族上给出了构造性版本。

先解释对象。图的 cut 是把顶点分成 $S$ 和 $V\setminus S$ 两边后，跨过这两边的边集，记作 $\delta(S)$。图是 $k$-edge-connected，意思是每个非平凡 cut 至少有 $k$ 条跨边；最小 cut 的大小就是 $k$。一棵 spanning tree 必须连通所有顶点，所以对每个 cut 至少要跨过去一次。Thin tree 问的是能不能选一棵生成树，让它对每个 cut 都“少拿边”：形式化地，$T$ 是 $\alpha$-thin，如果对所有 $\emptyset\subsetneq S\subsetneq V$，
$$
|T\cap \delta(S)|\le \alpha |\delta(S)|.
$$
最小 cut 给出一个硬下界：因为 $|\delta(S)|=k$ 而 $|T\cap\delta(S)|\ge 1$，所以不可能希望 $\alpha$ 比 $1/k$ 小到常数因子之外。Strong thin tree conjecture 正是说，每个 $k$-edge-connected 图都有 $C/k$-thin spanning tree，其中 $C$ 是绝对常数。

这个问题之所以长期有人关心，主要不是因为“生成树跨割少”本身，而是因为它把全局连通性问题压缩成一个很干净的离散结构。Asadpour et al. (2017) 在 ATSP 近似算法中提出强版本，并说明 strong thin tree conjecture 会推出 asymmetric traveling salesperson problem 标准 LP 的 $O(1)$ integrality gap。ATSP 后来由 Svensson, Tarnawski, and Végh (2020) 得到常数近似 [arXiv:1708.04215](https://arxiv.org/abs/1708.04215)，Traub and Vygen (2020) 又把近似比改进到 $22+\varepsilon$ [arXiv:1912.00670](https://arxiv.org/abs/1912.00670)，但 thin tree 仍然是一条不同的结构性路线：如果它成立，就说明“高度边连通图里总能抽出一棵对所有瓶颈都很轻的树”。它也和 Pritchard (2011) 的 removable spanning tree conjecture 有直接关系：是否总能删掉一棵生成树后，只损失 $o(k)$ 的边连通度；强版本要求只损失 $O(1)$。

这里的关系可以直接从公式看出来。如果 $T$ 是 $f(k)$-thin，那么删掉 $T$ 后，任意 cut 至少还剩
$$
|\delta(S)|-|T\cap\delta(S)| \ge |\delta(S)|(1-f(k))\ge k(1-f(k))
$$
条边，所以剩余图仍然是 $k-kf(k)$ edge-connected。强 thin tree conjecture 给 $f(k)=O(1/k)$，于是删树只损失 $O(1)$ 连通度。反过来，Pritchard conjecture 对大 cut 的要求其实更弱：如果一个 cut 有 $k^2$ 条边，删树后只需要还剩接近 $k$ 条跨边，并不要求生成树只占 $O(1/k)$ 的比例。真正难压住的是 minimum cuts 和 near minimum cuts，因为那里每多拿一条树边都会直接吃掉有限的连通性预算。本文正好解决这部分最紧的割。

已有结果离强 conjecture 还很远。Goddyn (2004) 提出 thin tree conjecture 的弱形式：存在 $f(k)\to 0$，使得每个 $k$-edge-connected 图有 $f(k)$-thin tree。Oveis Gharan and Saberi (2011) 证明 planar 和 bounded genus 图满足相应结论。一般图中，Anari and Oveis Gharan (2015) 的存在性结果给出 $O(\mathrm{poly}\log\log n/k)$-thin tree [arXiv:1411.4613](https://arxiv.org/abs/1411.4613)；Asadpour et al. (2017) 的构造性算法给出 $O(\log n/(\log\log n)\cdot 1/k)$ 级别的 thinness，但论文也提醒，这个随机算法并不完全满足“给出后可高效验证 thinness”的构造性要求，因为检查一棵树是否对所有 cut 都 thin 本身没有已知多项式测试。另一方面，如果只要求某些 cut family 上 thin，Klein and Olver (2023) 已经解决了 laminar family 的情形 [arXiv:2304.07674](https://arxiv.org/abs/2304.07674)。Laminar family 是一族没有交叉的集合：任意两个集合要么不相交，要么一个包含另一个。这种结构像一棵嵌套树，和任意 cut family 的指数级混乱相比要容易得多。

:::aside[Laminar thin tree 的输入形式]
Klein and Olver (2023) 的结果可理解为：给定 $k$-edge-connected 图 $G$ 和 laminar cut family $\mathcal L$，可以找 spanning tree $T$，使得对所有 $S\in\mathcal L$ 有 $|T\cap\delta(S)|\le O(|\delta(S)|/k)$。本文需要的加强形式是：若 $\mathcal L$ 中每个 cut 都有 $|\delta(S)|\le 4k/3$，则可保证 $|T\cap\delta(S)|\le 11$。论文在 appendix 里从 KO23 的 rounding theorem 中展开了这个常数。
:::

最自然的想法是：near minimum cuts 虽然不是 laminar，但能不能从中挑一个 maximal laminar subfamily，然后用 Klein-Olver 的 theorem？对真正的 minimum cuts，这类想法是可行的。Minimum cuts 有 cactus representation，且可以 uncross：如果 $A,B$ 是 crossing minimum cuts，那么 $A\cap B$ 和 $A\cup B$ 仍然是 minimum cuts。换句话说，minimum cuts 的交叉不是任意混乱，它背后有一个仙人掌结构支撑。

Near minimum cuts 就不同了。如果 $A$ 是 $\eta_A$-near minimum cut，$B$ 是 $\eta_B$-near minimum cut，且二者 crossing，那么 $A\cap B,A\cup B,A\setminus B,B\setminus A$ 只能保证是 $(\eta_A+\eta_B)$-near minimum cuts。这个参数会随着反复 uncrossing 累积，很快离开原来的 near-minimum 范围。论文给了一个很说明问题的例子：在一个多重 Hamiltonian cycle 上叠加另一条 Hamiltonian cycle，可以让大量区间都是 near minimum cuts；某个 maximal laminar family 上 thin 的树，可能在另一个 near minimum interval 上几乎把整棵树都压过去。所以本文的 reduction 不是“取一个足够大的 laminar 子族”，而是要构造一个特殊的 laminar family，使每个 near minimum cut 的边界都能被少数 laminar cuts 的边界覆盖。

主定理可以写成：
$$
\textbf{Theorem 1.1.}\quad
\begin{aligned}
&\text{Let }G=(V,E)\text{ be }k\text{-edge-connected. For }\eta=1/40,\\
&\text{there is a spanning tree }T\text{ such that every cut }S\\
&\text{with }|\delta(S)|<(1+\eta)k\text{ satisfies }|T\cap\delta(S)|\le 88.
\end{aligned}
$$
Moreover, such a tree can be found in polynomial time.

和 prior best 的定量比较要小心说。它没有改进 general all-cuts thinness 的 $O(\mathrm{poly}\log\log n/k)$ 存在性界，也没有给完整 strong thin tree conjecture。但在 near minimum cut family 上，它把 bound 做到了 conjectured scale $O(1/k)$，而且是 polynomial-time constructive。这个 cut family 本身并不小：cycle 上 near minimum cuts 可以有 $\Omega(n^2)$ 个，而 laminar family 最多 $2n-1$ 个。因此结果的核心不是“cut 数量少”，而是“cut 结构可被几何表示后，有一个常数覆盖的 laminar skeleton”。

论文的技术入口是 Benczúr-Goemans polygon representation。直觉上，它把一个 crossing-connected component 的 near minimum cuts 画成一个凸多边形里的对角线：原图顶点被分成若干 atoms，每个 atom 是一组在该 cut component 中永远不会被分开的原图顶点；每条代表性对角线把多边形两边的 atoms 分开，对应一个 near minimum cut。贴着多边形外边的 atoms 叫 outside atoms，内部 cell 里的叫 inside atoms。对 $\eta\le 1/5$ 的 near minimum cuts，每个 crossing component 都有这种 representation，且 crossing 关系可以在 outside atoms 的 cyclic order 上读出来。

这一步很关键，因为它把 cut family 的交叉结构从图论对象变成了区间对象。固定一个 root outside atom $a_0$ 后，其他 outside atoms 沿多边形边界排成 $a_1,\ldots,a_{n-1}$。任何不含 root 的 canonical cut 可以用一个区间 $\langle l,r\rangle=\{a_l,\ldots,a_{r-1}\}$ 表示。如果这个区间对应某个 $1/5$-near minimum cut，就叫 shadow；如果对应本文关心的 $\eta$-near minimum cut，就叫 near minimum interval。于是问题变成：在一个圆周区间系统里，构造 laminar intervals，使每个 near minimum interval 分开的 atom pairs 都被常数个 laminar intervals 分开。

:::aside[Polygon representation 的精确角色]
对一个 near-min-cut crossing component $\mathcal C$，atoms 是顶点集的最粗 partition，使得每个 $S\in\mathcal C$ 都是若干 atoms 的并。Polygon representation 把 atoms 放进凸多边形的 cells，并让 $\mathcal C$ 中每个 cut 对应一条 diagonal 的一侧。本文只在 $\eta=1/40$ 下运行算法，但会生成 $4\eta=1/10$-near minimum cuts；这仍小于 $1/5$，所以 polygon representation 和 KKO22 的 inside-atom lemma 可用。
:::

作者先定义 special intervals。一个 canonical cut 是 special，如果它本身是 $\eta$-near minimum cut，或者它可由两个 crossing 的 $\eta$-near minimum cuts 取交、并、差得到。由 near-min-cut 的 crossing lemma，special cut 是 $2\eta$-near minimum cut；两个 crossing special cuts 的差至多是 $4\eta$-near minimum cut。这个定义看起来只是为了常数预算服务，但它实际承担了一个闭包功能：如果 special interval $A$ 和 near minimum interval $B$ crossing，那么 $A\cap B$ 与 $A\setminus B$ 至少有一个 special，$A\cup B$ 与 $B\setminus A$ 至少有一个 special。证明只是区间位置分类，但这个小引理是后面递归覆盖能不断“补洞”的原因。

构造 laminar family 的算法叫 `MakeLam`。它在一个当前区间 $\langle L,R\rangle$ 上递归，深度为偶数时从左端出发，收集所有 special prefixes $\langle L,x\rangle$，把相应 cuts 加入 $\mathcal L$，再递归到相邻 prefixes 之间的环带；深度为奇数时对称地从右端出发，收集 special suffixes $\langle x,R\rangle$。如果某一层从当前方向找不到 special interval，就换方向；连续两个方向都找不到时，缩掉一个端点，保证递归推进。

这段算法最容易被误读成一个技术性贪心，但我觉得它的设计点很明确：它不是想把所有 near minimum intervals 放进 laminar family，而是要保证每个目标 interval 在递归树的某个最高包含节点附近，被两三个相邻层的 special cuts 从两侧夹住。假设目标是 $S=\mathrm{cut}(l_S,r_S)$，在包含它的最高递归区间 $\langle l_1,r_1\rangle$ 里，如果当前从左处理，就取
$$
i=\min\{x:l_S\le x\le r_S,\ \langle l_1,x\rangle\text{ is special}\},
$$
$$
j=\max\{x:l_S\le x\le r_S,\ \langle l_1,x\rangle\text{ is special}\}.
$$
算法已经加入了 $\mathrm{cut}(l_1,i)$ 和 $\mathrm{cut}(l_1,j)$。这两个 cut 会覆盖 $S$ 分开的绝大多数 outside atom pairs；剩下的 pair 只出现在左侧缺口 $\langle l_1,l_S\rangle$ 对 $\langle l_S,i\rangle$，以及右侧缺口 $\langle j,r_S\rangle$ 对外部区域。Special-interval closure lemma 说明这些缺口本身又能在下一两层递归中被 special cuts 或一个“两个 crossing special cuts 的差”覆盖。论文最后得到：对每个 near minimum cut $S$，存在至多 $8$ 个 $\mathcal L$ 中的 cuts，覆盖 $S$ 分开的所有 outside atom pairs。

这里“覆盖”不是集合包含，而是边界包含。论文定义 cut collection $\mathcal B$ covers $S$，如果
$$
\delta(S)\subseteq \bigcup_{B\in\mathcal B}\delta(B).
$$
在 atom 语言里，这等价于：任何被 $S$ 分到两侧的 atom pair，也被某个 $B\in\mathcal B$ 分到两侧。一旦有这个性质，对任意 spanning tree $T$，
$$
|T\cap\delta(S)|\le \sum_{B\in\mathcal B}|T\cap\delta(B)|.
$$
所以如果 $\mathcal L$ 中每个 cut 都能由 Klein-Olver theorem 控制到 $11$ 条树边，而 $|\mathcal B|\le 8$，就得到 $88$。

Outside atoms 的分析还不够，因为 polygon 内部可能有 inside atoms。论文这里用到 Karlin, Klein, and Oveis Gharan (2022) 关于 polygon representation 的一个性质：如果若干 diagonal 所围出的区域由少于 $1/(2\eta)$ 个 halfplanes 描述，且不碰多边形外边界，那么这个区域没有 inside atom。本文的 $\eta=1/40$ 让阈值是 $20$。在 outside coverage 的证明中，覆盖一个 $S$ 用到的 cuts 至多涉及 $19$ 条相关 diagonals：$S$ 自己一条，最多 $7$ 个 special cuts 各需两条 halfplane 边界，再加一个 set-difference cut 最多四条。$1+2\cdot 7+4=19<20$。因此这些 diagonals 划出的每个含 inside atom 的区域，必然也含某个 outside atom。把 inside atom 替换成同一区域内的 outside atom，所有相关 diagonals 对它们的分隔关系完全一样；outside pair 已经被覆盖，inside pair 也随之被覆盖。

这一步是全文我最喜欢的地方。前半段递归算法几乎完全在一维区间上运行，看起来像是把 polygon representation 的内部复杂性忽略了；最后用 $19<20$ 这样一个常数不等式把 inside atoms 一次性接回去。常数 $1/40$ 也在这里变得具体：它不是 thin tree 本身要求的自然阈值，而是为了让“生成的覆盖证书涉及的 halfplanes 数量”落在 KKO22 empty-region lemma 的适用范围内。作者也明确说没有优化常数；polygon representation 本身可到 $\eta\le 1/5$，但把本文方法推到这个上限需要更精细的覆盖或更少的 diagonals。

最后还要把不同 crossing components 的 laminar families 合并。论文先为每个 nontrivial crossing component 构造一个 cross-free family；孤立 near minimum cut 直接加入。Benczúr 的结构 lemma 说，不同 crossing components 的 atoms 中存在一对 $a,a'$ 满足 $a\cup a'=V$，这足以推出来自不同 components 的 cuts 不会 crossing。然后固定原图一个 root vertex $r$，对每个 cut 选择不含 $r$ 的一侧，就把 cross-free family 统一转成 laminar family $\mathcal L$。这一步看似形式化，但它保证了可以一次性调用 Klein-Olver 的 laminar thin tree theorem，而不是为每个 polygon component 分别找树。

完整算法于是很短：枚举所有 $\eta$-near minimum cuts 并计算其 crossing components 与 polygon representations（Benczúr 1995 给出多项式算法）；对每个 polygon 运行 `MakeLam` 得到 laminar cuts；合并成全局 laminar family $\mathcal L$；调用 Klein-Olver (2023) 的 laminar-family spanning-tree rounding，得到一棵树 $T$，使每个 $B\in\mathcal L$ 满足 $|T\cap\delta(B)|\le 11$。对任意原始 near minimum cut $S$，若它是孤立 cut，直接有 $11$；否则由 Theorem 3.1，它被至多 $8$ 个 $\mathcal L$-cuts 覆盖，因此
$$
|T\cap\delta(S)|\le 8\cdot 11=88.
$$

这篇论文给 strong thin tree conjecture 的启发不是“near minimum cuts 已经足够接近全部 cuts”，而是展示了一个更局部的可能路线：先识别最受约束的 cut family，把它规约成 laminar rounding 能处理的对象，再考虑大 cuts 需要怎样的弱 thinness。对 Pritchard removable spanning tree conjecture 来说，这正是正确的分工。Minimum 和 near-minimum cuts 需要 $O(1)$ 条树边；远大于 $k$ 的 cut 只需要树别把它吃空。本文把第一部分做成了一个真正可构造的结构定理，剩下的问题不再是“所有 cut 同时 $O(1/k)$-thin”这么硬，而是能否为大 cut 找到与 removable tree 目标匹配的更粗控制。

## Low-Cost Arborescence Under Edge Faults [arXiv:2605.13800](https://arxiv.org/abs/2605.13800)

这篇论文研究的是 directed graph 里的 min-cost arborescence 在单条边临时失效时如何快速维护：给定有向图 $G=(V,E)$、根 $r$ 和非负边成本 $\mathrm{cost}:E\to \mathbb{R}_{\ge 0}$，先算出一棵以 $r$ 为根的最小成本 arborescence $T$；之后每次只有一条边 $f$ 失效，故障修复后又回到原图。作者证明，可以预处理出一个只有 $O(n^{3/2})$ 条边的子图 $H$，使得对任意边故障 $f$，在 $H-f$ 中求到的 min-cost arborescence 是 $G-f$ 中最优值的 $2$-approximation。换句话说，故障发生时不必在原来的 $m$ 边图上重新跑 Gabow-Galil-Spencer-Tarjan 算法，而是在稀疏子图上花 $O(n^{3/2})$ 时间得到 $2$ 倍近似。

论文还处理了一个 matroid 版本：给定 matroid $M=(E,\mathcal{I})$ 和元素成本，希望预先保留一个小集合 $S\subseteq E$，使得任意至多 $k$ 个元素失败后，$M|(S\setminus F)$ 里仍然含有 $M|(E\setminus F)$ 的 min-cost basis。作者给出一个极其直接的 greedy-layer construction，并声称得到 tight 的 $k\cdot \mathrm{rank}(E)$ 型 preserver；正文算法实际取了 $k+1$ 层 disjoint min-cost bases 来处理至多 $k$ 个 failures，所以更自然的参数写法是“至多 $k$ 个 failures 需要 $k+1$ 层”。这看起来是论文表述中的 off-by-one 约定或笔误，但技术内容本身很清楚：故障可以沿着这些 basis layers 一层层做 exchange。

先解释第一个对象。Arborescence 可以理解成有向版的生成树：除了根 $r$ 没有入边以外，每个顶点恰好有一条入边，并且从 $r$ 能沿有向边到达所有顶点。它和 shortest-path tree 很不一样。Shortest-path tree 关心的是从 $r$ 到每个点的距离都尽量短；min-cost arborescence 关心的是整棵树所有边的总成本最小。前者是逐点目标，后者是全局组合目标。这个区别在故障维护里非常重要，因为一条替换边可能让某个点路径变长，却显著降低整棵树的总成本。

为什么要研究这类 sparse fault-tolerant subgraph？直接方案当然是：对每条可能故障的树边 $f\in T$，预先把 $G-f$ 的最优 arborescence $T_f$ 都算出来并存下来。问题是这要存 $n-1$ 棵树，总空间 $\Theta(n^2)$。另一种直接方案是故障发生时在 $G-f$ 上重新计算，但这要 $O(m+n\log n)$ 时间，在 dense graph 中太慢。Sparse preserver/subgraph 的思想介于两者之间：预处理时不存答案表，而是存一个小子图 $H$；在线阶段只在 $H-f$ 上求解。这个框架在 graph algorithms 里很常见，尤其适合“网络大、故障少、查询反复发生”的场景。

已有脉络里，最接近的是 unweighted single-source reachability。若所有边成本都为 $1$，arborescence 的成本只是 $n-1$，问题退化为：故障后哪些点仍能从源点到达，能否由一个稀疏子图精确保留？Baswana, Choudhary, and Roditty (2018) 证明了对 $k$ 条边故障存在 $O(2^k n)$ 边的 fault-tolerant reachability subgraph，并且这个界是 tight；后续 Lokshtanov et al. (2019) 给出基于 important separators 的更简洁证明 [arXiv:1904.08150](https://arxiv.org/abs/1904.08150)。在 unweighted BFS/shortest-path 方向，Parter and Peleg (2013) 证明单故障 FT-BFS tree 可以做到 $O(n^{3/2})$ 边且有匹配下界 [arXiv:1302.5401](https://arxiv.org/abs/1302.5401)。但是一旦进入 weighted shortest paths，精确保留就迅速变得昂贵：Demetrescu et al. (2008) 的 replacement distance oracle 线表明加权有向图中 fault-avoiding distances 本身需要接近二次级别的空间才能精确快速回答。

这就是本文结果的位置：weighted directed graph 通常是 sparse exact preserver 很难存在的地带，而 min-cost arborescence 居然能在单边故障下得到 $O(n^{3/2})$ 边、$2$ 倍近似的 universal subgraph。这里“universal”指 $H$ 同时服务所有可能的单边故障，而不是为某个固定 $f$ 临时构造。

形式化地，论文的第一个主定理是：

$$
\textbf{Theorem 1.1.}\quad
\text{对任意带非负成本的有向图 }G=(V,E),\ |V|=n,
\text{存在子图 }H\subseteq G,\ |E(H)|=O(n^{3/2}),
$$

$$
\text{使得对任意 }f\in E,\quad
\mathrm{OPT}(H-f)\le 2\cdot \mathrm{OPT}(G-f),
$$

其中 $\mathrm{OPT}(X)$ 表示图 $X$ 中 min-cost arborescence 的成本。构造时间为 $O(mn+n^2\log n)$；故障发生后，在 $H-f$ 上运行 min-cost arborescence 算法即可，时间变成 $O(n^{3/2})$ 量级。

这个算法简单到有点出人意料。先计算一棵原图 $G$ 的 min-cost arborescence $T$。对 $T$ 中每条边 $e_i=(p(v_i),v_i)$，删掉它后，$T-e_i$ 分成两块：包含根 $r$ 的部分记为 $X(v_i)$，以 $v_i$ 为根的断开子树记为 $T(v_i)$。为了重新接回 $v_i$，作者在 $G-e_i$ 中找一条从 $X(v_i)$ 到 $v_i$ 的 shortest path，记为 $\rho_i$。最后令

$$
H = T \cup \bigcup_{i=1}^{n-1}\rho_i.
$$

注意这里 shortest path 的权重就是原始边成本，而不是某种 arborescence 的 reduced cost。这一点决定了它只能得到 $2$-approximation，而不是 exact preserver。

为什么 $2$ 倍近似成立？设故障边正是 $f=e_i=(p(v_i),v_i)$。在 $H-f$ 里，$T-f$ 加上路径 $\rho_i$ 已经给出一棵可行 arborescence，因此 $H-f$ 的最优解 $A_f$ 不会比它更贵：

$$
\mathrm{cost}(A_f)\le \mathrm{cost}(T)+\|\rho_i\|.
$$

而 $\rho_i$ 是从 $X(v_i)$ 到 $v_i$ 的最短路，且 $r\in X(v_i)$，所以 $\|\rho_i\|$ 不超过 $G-f$ 中从 $r$ 到 $v_i$ 的最短路长度。任意 $G-f$ 中的 arborescence 都必须包含一条从 $r$ 到 $v_i$ 的路径，因此

$$
\|\rho_i\|\le \mathrm{OPT}(G-f).
$$

同时 $T$ 是原图最优，删边只会让问题更受限，所以 $\mathrm{cost}(T)\le \mathrm{OPT}(G-f)$。合起来就是

$$
\mathrm{cost}(A_f)\le \mathrm{cost}(T)+\|\rho_i\|
\le 2\cdot \mathrm{OPT}(G-f).
$$

这段证明里真正省力的地方，是把“重连断掉的子树”只压到一个顶点 $v_i$ 上，而不是试图为整个子树重新优化。最优 arborescence 的全局结构很复杂；但要得到 $2$ 倍，只要知道任何可行解必须把 $v_i$ 从根连通起来，这个必要条件已经够用。

难点剩在稀疏性：$n-1$ 条 replacement paths，每条可能有 $\Theta(n)$ 条边，朴素并集是 $\Theta(n^2)$。作者证明它们的并集只有 $O(n^{3/2})$ 条边。这个部分借鉴了 distance preserver 里常见的 path-union charging，但有一个麻烦：这些 $\rho_i$ 不是同一张图里的普通 shortest paths，而是分别在 $G-e_i$ 里算出来的 replacement shortest paths。也就是说，路径集合来自不同的故障图。

证明做了一个 coloring。按 $\rho_1,\rho_2,\ldots,\rho_{n-1}$ 的顺序加入路径；一条边第一次出现在哪条路径里，就给它染上对应颜色 $i$。设颜色 $i$ 的边数为 $c_i$。如果能控制 $\sum_i c_i^2$，Cauchy-Schwarz 就给出

$$
\sum_i c_i \le \sqrt{n\sum_i c_i^2}.
$$

因此目标变成证明 $\sum_i c_i^2=O(n^2)$。

作者把颜色 $i$ 的边 charge 到有序点对 $(x,y)$。直觉上，如果 $\rho_i$ 里从 $x$ 到 $y$ 的子路径中，$x$ 有一条新染成 $i$ 的出边，$y$ 有一条新染成 $i$ 的入边，就把颜色 $i$ charge 给 $(x,y)$。一条含 $c_i$ 条新边的路径会 charge 约 $c_i(c_i+1)/2$ 个点对，所以

$$
\sum_i c_i^2
\le 2\sum_{(x,y)}\#\{\text{charged colors for }(x,y)\}.
$$

核心引理是：每个点对 $(x,y)$ 最多被三个颜色 charge。因为总点对只有 $n^2$ 个，于是 $\sum_i c_i^2\le 6n^2$，最终得到 $\sum_i c_i\le \sqrt{6}n^{3/2}$。

:::aside[唯一最短路假设]
论文为简化论证假设任意删边集合 $F$ 后，任意两点间 shortest path 都唯一。标准做法是对边成本做 infinitesimal random perturbation 或按 lexicographic tie-breaking 选路。这个假设不改变近似结构，主要用于避免“两个同长替换路径交错”时的记号复杂度。
:::

为什么一个点对最多被三个颜色 charge？固定 $(x,y)$，看原图 $G$ 里的唯一 shortest path $\mathrm{SP}(x,y)$。对于某条 $\rho_i$ 的子路径 $\rho_i[x,y]$，分两类：它和 $\mathrm{SP}(x,y)$ 边不相交，称为 non-intersecting；否则称为 intersecting。若两条 replacement subpaths 都和 $\mathrm{SP}(x,y)$ 不相交，那么它们必须相同。原因是每条这样的子路径都是“删掉某条位于 $\mathrm{SP}(x,y)$ 上的故障边后的 replacement shortest path”；若两条不同，就能用唯一最短路在删掉两条边后的图中推出矛盾。因此 non-intersecting 颜色最多贡献一次 charge。

Intersecting 的情况更细。令 $(x,a)$ 和 $(b,y)$ 分别是 $\mathrm{SP}(x,y)$ 的第一条和最后一条边。作者证明：任何 intersecting 的 replacement subpath $\rho_\ell[x,y]$，若要从 $x$ 到 $y$ 又和 $\mathrm{SP}(x,y)$ 相交，那么它必须包含 $(x,a)$ 或 $(b,y)$ 至少之一。否则可以在某个交点 $z$ 处把 $\rho_\ell$ 的一半和原 shortest path 的另一半拼起来，得到一条在 $G-e_\ell$ 中更短的 $x\to y$ 路径，矛盾。现在 $(x,a)$ 第一次出现时已经被某个颜色染掉，$(b,y)$ 第一次出现时也被某个颜色染掉；之后的 intersecting 路径即便经过这些边，也无法再让它们成为“颜色 $\ell$ 的新出边/新入边”。所以 intersecting 颜色最多两个。一个 non-intersecting 加两个 intersecting，正好是三。

我觉得这部分最好的地方是：近似分析和稀疏性分析完全分离。$2$ 倍近似只需要“一条从根侧接回断点的短路径”；而 $O(n^{3/2})$ 稀疏性只看这些 replacement paths 的几何重叠结构。两者各自都不复杂，放在一起却绕开了 weighted directed shortest-path preserver 里常见的二次空间障碍。

第二个结果切到 matroid。Matroid 是把“线性无关”“森林”“匹配型可选集合”等结构抽象到同一个 exchange axiom 下的语言。形式上，matroid $M=(E,\mathcal{I})$ 有一个 ground set $E$ 和一族 independent sets $\mathcal{I}$，满足向下封闭性以及 exchange property：若 $X,Y\in\mathcal{I}$ 且 $|Y|>|X|$，则存在 $e\in Y\setminus X$ 使得 $X\cup\{e\}\in\mathcal{I}$。Basis 是 maximal independent set；所有 basis 大小相同，称为 rank。

在 matroid 里，min-cost basis 可以由 greedy algorithm 求出：按成本从小到大扫描元素，能加入且保持 independent 就加入。这个事实是 matroid 最核心的算法性质。本文的 fault-tolerant preserver 问题问：能否预先保留少量元素，使得任意 failures 后，幸存的保留集合里仍然包含全局最优的 min-cost basis？

论文定义的 $k$-fault tolerant preserver 是一个集合 $S\subseteq E$，满足对所有 $F\subseteq E$ 且 $|F|\le k$，

$$
M|(S\setminus F)\text{ contains a min-cost basis of }M|(E\setminus F).
$$

这里 $M|X$ 是 restriction，也就是只允许使用 $X$ 中元素的 matroid。和前面 arborescence 部分不同，这里要求的是 exact preserver：失败后，$S\setminus F$ 里真的有全局 min-cost basis，而不是近似解。

构造非常自然。先在 $E$ 上取 min-cost basis $B_1$；删掉 $B_1$ 后，在 $E\setminus B_1$ 上取 min-cost basis $B_2$；继续下去，取若干个 disjoint greedy bases，最后令

$$
S = B_1\cup B_2\cup\cdots\cup B_{k+1}.
$$

若要处理至多 $k$ 个 failures，这个 $k+1$ 层写法正好对应证明中的“每发生一个 failure，就消耗掉最底下一层的一个交换机会”。论文 Theorem 1.3 表述为大小 $k\cdot \mathrm{rank}(E)$ 的 tight preserver，但 Algorithm 2 和后续 multiple failures 论证使用的是 $B_1,\ldots,B_{k+1}$；因此读这一定理时应把重点放在“线性于故障数和 rank，且由 consecutive greedy bases 实现”这个结构结论上。

先看单个 failure。设 $B_1$ 是原 matroid 的 min-cost basis，元素 $f\in B_1$ 失败。如果 $\mathrm{rank}(E-f)<\mathrm{rank}(E)$，那 $B_1-f$ 已经是幸存 matroid 的 basis，问题结束。否则，论文先证明存在某个元素 $x\in E-f$，使得 $B_1-f+x$ 是 $M|(E-f)$ 的 min-cost basis。这可以看成 matroid greedy 的 sensitivity lemma：删掉一个已选元素后，新的最优 basis 可以通过一次 swap 得到。

但 $x$ 未必在第二层 $B_2$ 里。关键是证明 $B_2$ 中一定有某个不比 $x$ 贵、且同样能修好 $B_1-f$ 的元素。令 $b_1,\ldots,b_\ell$ 是所有满足 $B_2-b_i+x$ 仍为 $E\setminus B_1$ 上 basis 的元素。Matroid exchange 给出

$$
x\in \mathrm{span}(\{b_1,\ldots,b_\ell\}).
$$

又因为 $B_2$ 是 $E\setminus B_1$ 上的 min-cost basis，$B_2$ 不会比 $B_2-b_i+x$ 更贵，所以每个相关的 $b_i$ 都满足 $\mathrm{cost}(b_i)\le \mathrm{cost}(x)$。如果所有这些 $b_i$ 都落在 $\mathrm{span}(B_1-f)$ 里，那么 $x$ 也会落在 $\mathrm{span}(B_1-f)$ 里，和 $B_1-f+x$ 是 basis 矛盾。因此某个 $b_i$ 可以替代 $f$，且成本不高于 $x$，于是 $B_1-f+b_i$ 也是 min-cost basis。

多 failure 的证明就是把这个 swap 递归化。第一次故障让一个元素从 $B_2$ 移到 $B_1$；这相当于 $B_2$ 自己“损失”了一个元素，于是再从 $B_3$ 找 swap element 补给 $B_2$。每处理一个真实 failure，整条 basis chain 往上做一次局部交换，消耗掉最底层一层。这个画面比形式证明更容易记：$B_1$ 是当前要维护的最优 basis，$B_2,B_3,\ldots$ 是按成本顺序准备好的 spare layers；matroid exchange axiom 保证 spare layer 里的元素可以被逐级推上来，而且不会破坏 min-cost 性。

这和 Bentert et al. (2025) 的 fault-tolerant matroid basis 问题不同 [arXiv:2506.22010](https://arxiv.org/abs/2506.22010)。他们研究的是找一个最小集合 $B$，使得删掉其中任意 $k$ 个元素后仍能 span 整个 matroid，重点是 feasibility/robust spanning，且一般情形 NP-hard、给出 FPT 算法。本文的 preserver 问题则是：失败集合可以来自整个 $E$，保留集合要保证失败后仍含有 min-cost basis；目标不是自己在失败后仍 spanning，而是保留足够多的“全局最优候选元素”。这个区别解释了为什么这里会出现一个干净的 greedy-layer 定理。

Arborescence 部分最后没有给 exact preserver，而附录说明了为什么不是把路径权重改成 reduced/effective costs 就能解决。若要精确维护 $G-f$ 的 min-cost arborescence，可以把每条非树入边的成本改成相对当前树入边的 effective cost；负 effective cost 的“向上边”可能让一条看似更贵的接回路径整体更优，因为它允许之后加入若干负成本边来降低总成本。这样一来，最短接回路径的局部最优性不再足以控制全局 arborescence 成本，本文用来证明 $O(n^{3/2})$ 的 replacement shortest path 结构也就失效了。

所以这篇论文的边界很清楚：如果愿意接受 factor $2$，weighted directed arborescence 在单边故障下有一个非常轻的 sparse subgraph；如果坚持 exact，问题立刻暴露出 directed branching 的全局依赖。这个对比也是本文最有信息量的地方。它不是简单地把 unweighted reachability 的 FTRS 搬到 weighted setting，而是找到一个刚好能让 shortest-path replacement 和 arborescence objective 接上的近似层级。

## 未入选但值得关注



### Distributed Approximate Maximum Matching and Minimum Vertex Cover via Generalized Graph Decomposition [arXiv:2605.13264](https://arxiv.org/abs/2605.13264)

这篇有 PODC 2026 背书，并给出用于分布式 approximate matching 与 vertex cover 的 generalized graph decomposition。今天没有选它，主要是因为四篇已有完整解读的稿件里，learning、quantum 和两篇图算法的基础问题定位更适合深讲；这篇更偏特定 round bound 的算法推进。



### On the Complexity of the Minimum-($k,\rho$)-Shortcut Problem [arXiv:2605.13474](https://arxiv.org/abs/2605.13474)

论文给 minimum shortcut 问题几乎完整的复杂性边界，属于很明确的复杂性分类工作。未入选是因为主题相对窄，需要较多篇幅铺设 shortcut 模型，今天版面优先给影响面更宽的四篇。



### On the Advantage of Adaptivity for Sampling with Cell Probes [arXiv:2605.12873](https://arxiv.org/abs/2605.12873)

题目显示它可能给出 sampling in the cell-probe model 中 adaptivity 优势的下界或分离，方向值得关注。当前缓存与工具没有提供可靠摘要，无法确认具体模型和主定理，所以不适合在日报里深讲。



### Min-Max Optimization Requires Exponentially Many Queries [arXiv:2605.13806](https://arxiv.org/abs/2605.13806)

这篇看起来是 min-max optimization 的黑盒 query lower bound，若模型足够标准会有较强理论意义。未入选原因同样是当前缓存缺少摘要，优化/博弈交叉论文又高度依赖 oracle 和 promise 细节，贸然解读风险较高。



### LFPL: Revisited and Mechanized [arXiv:2605.12893](https://arxiv.org/abs/2605.12893)

论文对 LFPL 做自包含重述并给出 mechanized metatheory，形式化质量可能很扎实。今天没有选，是因为它更偏整理与机械化验证，原创理论推进不如入选论文突出。



### Provable Quantization with Randomized Hadamard Transform [arXiv:2605.13810](https://arxiv.org/abs/2605.13810)

这篇给 randomized Hadamard transform quantization 的严格 MSE 保证，应用面不错。未入选是因为它更接近 ML systems primitive 的理论分析，和今日 TCS 深讲主线相比优先级略低。
