## 编辑按语

今天手里只有一篇完整解读，方向是 streaming algorithms。论文讨论 turnstile streaming algorithms 与 linear sketches 在 polynomial-length streams 下的等价性：当动态频率向量由增删更新给出、算法只能用小空间维护状态时，非线性 streaming 算法在多大程度上可以被线性测量替代。

这条问题线本身很核心，因为许多 streaming 下界只会先对 linear sketches 证明，再希望转回一般算法。本文把 Li--Nguyen--Woodruff 的长流等价和 Kallaugher--Price 的短流分离之间的空白往前推了一大步：在多项式长度这个最常用的范围内，turnstile 算法仍然基本等价于 linear sketch，证明工具则从 transition graph mixing 换成 Fourier analysis 和 additive combinatorics。

## Turnstile Streaming Algorithms Might (Still) as Well Be Linear Sketches, for Polynomial-Length Streams [arXiv:2604.22052](https://arxiv.org/abs/2604.22052)

这篇论文研究的是 turnstile streaming algorithms 和 linear sketches 的等价性：给定一个动态频率向量 $x\in\mathbb{Z}^n$，数据流由坐标更新 $(i,\Delta)$ 组成，$\Delta\in\{+1,-1\}$，算法只用很小空间维护状态，最后要从最终向量 $x$ 上输出某个函数或近似答案。Jiang, Liu, and Yu 证明，在 polynomial-length streams 这个最常见的长度范围内，如果一个 turnstile streaming algorithm 用 $S$ bits 空间并能处理长度 $\mathrm{poly}(D,n)$ 的流，那么在 $\|x\|_2\le D$ 的输入上，它可以由 $O(S)$ 个线性测量恢复，整体空间 $O(S\log S)$；在满足平均 smoothness 的近似问题上，还能得到 entry-bounded、维度 $O(S/\log D)$、总空间 $O(S)$ 的 sketch。

这里的 linear sketch 是指预先选一个矩阵或群同态 $A$，只保存 $Ax$，最后由 decoder 从 $Ax$ 输出答案。它是 streaming 里最常见的算法模板：比如频率矩、$\ell_p$ norm、heavy hitters、动态图中的 spanning forest 和 spectral sparsifier，很多最强算法本质上都是维护若干线性投影。问题在于，turnstile algorithm 本身并没有要求线性：它可以在每次更新后对内存做任意非线性操作。所以这条线的核心问题是：linear sketch 只是我们会设计，还是它其实已经覆盖了所有小空间 turnstile 算法？

Streaming model 存在的动机很直接：输入太大，不能存下来，也往往不能多遍扫描。比如网络包计数、数据库增删、动态图边更新，系统看到的是一串局部修改，最后关心的是某个全局统计量。Insertion-only streaming 只允许增加；turnstile model 允许增加和删除，因此更贴近动态维护。形式化地，unit update turnstile stream 是
$$
(i_1,\Delta_1),\ldots,(i_T,\Delta_T),\qquad i_t\in[n],\ \Delta_t\in\{+1,-1\},
$$
最终频率向量为 $x=\sum_t \Delta_t e_{i_t}$。如果要求每个前缀向量都在 $\mathbb{Z}_{\ge 0}^n$，就是 strict turnstile。算法空间小意味着它无法记住整个更新序列，只能保留一个压缩状态。

Linear sketch 的特殊之处是它完全忘记顺序，只看最终向量的线性像。若 $A$ 是 $m\times n$ 矩阵，算法维护 $Ax$；每次坐标更新只需把第 $i$ 列加上 $\Delta$ 倍。这个模板天然适合 turnstile，因为删除只是线性地减掉同一列。更重要的是，linear sketch 下界通常更容易证：可以转成 communication complexity，或者直接分析随机矩阵、整数矩阵、Gaussian measurement 的维度。因此，如果一般 turnstile 算法都能转成 comparable linear sketch，那么很多只对 sketch 证明的下界就自动变成 streaming 下界。

这正是 Li, Nguyen, and Woodruff (2014) [arXiv:1404.7585](https://arxiv.org/abs/1404.7585) 的 landmark result 所做的事。他们建立了一个概念性很强的等价：任意 $S$-space turnstile algorithm 可以模拟为 modular linear sketch。Ai, Hu, Li, and Woodruff (2016) [arXiv:1511.01778](https://arxiv.org/abs/1511.01778) 后续把它扩展到 strict turnstile、多遍算法和 bounded intermediate states。这个结果后来成了许多下界证明的入口：先把 streaming algorithm 变成 sketch，再调用 SMP communication、rank/minrank 或 real sketch lower bound。

但这个等价有一个很大的代价：它要求算法能承受极长的 stream，长度至少 doubly exponential in $n$ 或相关状态参数。原证明把 streaming algorithm 看成有限自动机，在 transition graph 上插入很长随机游走，让状态在强连通分量中混合到接近平稳分布。这个方法非常组合化，也非常依赖 uniform transition rule，即每一步使用同一更新函数。于是它不适用于 Read-Once Branching Programs 这种非均匀模型：第 $t$ 层可以有自己的转移规则。更现实的问题是，实际 streaming 算法关心的是 polynomial-length streams，不是双指数长度。

另一边，Kallaugher and Price (2020) [arXiv:2007.11516](https://arxiv.org/abs/2007.11516) 证明，短流下等价确实会失败。他们给出 linear-length streams 上的指数分离：有些问题存在 $O(\log n)$ 空间的非线性 streaming algorithm，但任何 linear sketch 都需要大得多的空间。这个结果说明 LNW14 的长流条件不是纯粹证明瑕疵。于是图景中间留下一个很自然的空白：线性长度可以分离，双指数长度可以等价，那么 polynomial length 到底站在哪一边？

Jiang, Liu, and Yu 这篇论文的答案是：在标准 turnstile 设定下，polynomial-length streams 仍然站在线性 sketch 一边。主结果的 exact 版本可概括为：
$$
\textbf{Theorem 1.1.}\quad
\begin{aligned}
&\text{若一个 }S\text{-bit turnstile algorithm 在所有长度 }\mathrm{poly}(R)\text{ 的流上}\\
&\text{以高概率计算 }f(x),\ R\ge \mathrm{poly}(D,n),\\
&\text{则对所有 }\|x\|_2\le D,\ f(x)\text{ 可由一个 randomized linear sketch 计算，}\\
&\text{sketch dimension 为 }O(S),\text{总空间为 }O(S\log S).\\
&\text{若 }S=O(\log R),\text{总空间可改进到 }O(S).
\end{aligned}
$$
论文在 Section 6 给出的正式陈述覆盖 approximation problems、promise problems 和一般 relation problems，而不只是单值函数。这个覆盖面很重要，因为 KP20 也有一个对 total functions 的短流 deterministic reduction，但它依赖“同一状态必须对应唯一正确答案”，不能直接处理近似输出或 promise/relation 类问题。

对于 smooth approximation problems，作者给出更强的 mollified transfer。这里 smooth 的意思不是函数可微，而是目标答案对小的离散高斯扰动稳定：若 $Y$ 来自目标输入分布 $I$，$Z\sim\gamma_R$ 是半径 $R$ 的离散高斯，则 $f(Y+Z)$ 通常与 $f(Y)$ 很接近。定理形式是：
$$
\textbf{Theorem 1.2.}\quad
\begin{aligned}
&\Pr_{Y\sim I,Z\sim\gamma_R}\big[|f(Y+Z)-f(Y)|\le \epsilon\big]\ge 1-\delta,\\
&\text{且存在 }S\text{-bit streaming algorithm 在长度 }\mathrm{poly}(R)\text{ 的流上}\\
&\text{以误差 }\epsilon\text{ 和成功率 }1-\delta\text{ 计算 }f,\\
&\text{则存在 entry-bounded integer linear sketch，维度 }O(S/\log R),\\
&\text{总空间 }O(S),\text{在 }I\text{ 上以 }1-O(\delta)\text{ 成功。}
\end{aligned}
$$
这个版本的定量强度来自维度而不只是 bit 数。很多 real-valued sketch lower bounds 证明的是测量数 $\Omega(d)$，而不是编码 bit 数；如果 transfer 只给 $O(S)$ rows 但每行信息量很小，就不能无损推出 $\Omega(d\log R)$ bits 的 streaming lower bound。mollified theorem 正好把 entry bound 和 dimension 同时控制住。

:::aside[Read-Once Branching Program]
Read-Once Branching Program (ROBP) 可以看成非均匀 streaming algorithm：输入流长度固定为 $T$，第 $t$ 层读取第 $t$ 个更新并把状态转到下一层，每层转移函数可以不同。普通 uniform streaming algorithm 若额外记住时间步，也能模拟长度 $T<2^{O(S)}$ 的 ROBP；但在更长流上，非均匀性不能免费消除。
:::

证明最让我意外的是它完全绕开了 transition graph mixing。作者不是试图把有限自动机混到平稳态，而是设计一个“noisy landing experiment”来暴露算法状态中真正保留下来的线性信息。先抽象成 decision problem $f:\mathbb{Z}^n\to\{0,1\}$。随机取 $M=\mathrm{poly}(n)$ 个大半径离散高斯增量 $X_1,\ldots,X_M\sim\gamma_R$，让 streaming algorithm 先处理这些 noisy prefix；若最终想测试目标 $y$，再追加一个更新块 $y-\sum_i X_i$，使最终频率向量正好落在 $y$。

算法只用 $S$ bits。记它处理第 $i$ 个 noisy block 后的状态为 $\sigma_i$，固定一条典型状态序列 $\sigma=(\sigma_1,\ldots,\sigma_M)$。由于算法 read-once，条件在整个状态序列上之后，各个 block 的 posterior laws
$$
\mu_i = \mathcal{L}(X_i\mid \sigma)
$$
仍然独立。因此 noisy prefix 总和的条件分布是卷积
$$
\nu=\mu_1*\cdots *\mu_M.
$$
从固定状态 $\sigma$ 出发，落到 $y_1$ 和落到 $y_2$ 所需的最后更新分别服从两个平移分布。如果 $v=y_1-y_2$ 满足某个 sketch kernel 条件，并且 $\nu$ 与 $\tau_v\nu$ 在 total variation distance 下很接近，那么算法几乎无法区分这两个目标；若算法在两者上都正确，就应当给出一致答案。于是要构造的 sketch $A$ 的任务变成：把所有让 $\nu$ 对平移敏感的方向记录下来，使得 $Av=0$ 时 $\nu$ 对 $v$ 平移近似不变。

Fourier analysis 在这里自然出现。对 $\mathbb{Z}^n$ 上的分布 $\mu$，Fourier transform 是定义在 torus $\mathbb{T}^n=[-1/2,1/2)^n$ 上的函数
$$
\widehat{\mu}(\zeta)=\sum_{x\in\mathbb{Z}^n}\mu(x)e^{-2\pi i\langle \zeta,x\rangle}.
$$
卷积在 Fourier 侧变成乘法，因此 $\widehat{\nu}(\zeta)=\prod_i \widehat{\mu_i}(\zeta)$。如果某个频率 $\zeta$ 的 Fourier coefficient 很大，直觉上说明分布沿着字符 $x\mapsto e^{2\pi i\langle \zeta,x\rangle}$ 有明显偏好，也就是质量集中在若干与 $\zeta$ 正交的周期性 slices 上。这样的频率就是平移敏感性的来源：若 $\langle \zeta,v\rangle$ 不是整数，平移 $v$ 会改变相位。

这给出两个典型障碍。几何障碍是 $\mu_i$ 可能都集中在某个低维子空间 $W$ 附近，那么垂直于 $W$ 的平移会把分布移出支撑；sketch 必须记录 $W^\perp$。算术障碍是 $\mu_i$ 可能只落在偶格点上，比如满足 $\sum_j x_j\equiv 0\pmod 2$，那么奇方向平移会让支撑错开；sketch 必须记录这个 parity constraint。Fourier 频率同时看见这两类障碍：前者是实线性方向，后者是 torus 上的有理频率。

关键结构引理说，$S$ bits 的状态不可能制造太多独立的 heavy frequencies。固定状态后，posterior law 是离散高斯的 dense piece，即大致满足 $\mu\le 2^{O(S)}\gamma_R$。如果存在很多彼此 dissociated 的 heavy frequencies，那么每个频率都把分布压到一组 slices 上，很多独立 slices 的交会在高斯测度下会指数变小；但 $\mu$ 相对于 $\gamma_R$ 的密度最多只有 $2^{O(S)}$，所以独立约束数量只能是 $O(S)$。论文把这个直觉做成了 Chang large-spectrum 风格的加性组合论论证：
$$
\textbf{Lemma 2.3.1 / Theorem 4.2.5.}\quad
\mu\le 2^{O(S)}\gamma_R,\ |\widehat{\mu}(\zeta_i)|\ge 1/2,
\text{ 且 }\{\zeta_i\}\text{ 为 }O(\sqrt S/R)\text{-dissociated}
\Rightarrow |\{\zeta_i\}|=O(S).
$$

:::aside[Dissociated frequencies]
一组 torus frequencies $A=\{\xi_1,\ldots,\xi_m\}\subset\mathbb{T}^n$ 称为 $\kappa$-dissociated，如果任意非零 $\varepsilon\in\{-1,0,1\}^m$ 都满足 $\|\sum_i\varepsilon_i\xi_i\|_{\mathbb{T}^n}\ge \kappa$。它是“没有短的加性关系”的近似版本，作用类似 additive combinatorics 中控制 large spectrum 的独立集。
:::

从这个引理到 sketch rows 还差一步：heavy frequencies 不是一个线性空间，而是在 torus 上带有模整数关系的集合。论文的 exact route 做了一个贪心 lattice construction。每当发现一个 heavy frequency $\zeta_j$ 远离已有生成元的组合，就加入一条链
$$
\zeta_j,\ q\zeta_j,\ q^2\zeta_j,\ldots
$$
直到继续扩展会破坏 dissociation。停止时得到一个近似关系 $k_j\zeta_j\approx v_j$，其中 $v_j$ 由旧频率生成。之后作者递归地把 $\zeta_j$ round 成精确生成元 $t_j$，满足
$$
k_jt_j\equiv \sum_{i<j}c_it_i \pmod{\mathbb{Z}^n}.
$$
sketch 记录的是 $\langle t_j,x\rangle\pmod{\mathbb{Z}}$，第 $j$ 行贡献约 $\log k_j$ bits。所有链的总长度由 dissociation bound 控制在 $O(S)$，从而总维度 $O(S)$；但为了控制 rounding error 的传播，目前 exact route 的总空间是 $O(S\log S)$，不是完全 tight 的 $O(S)$。

一旦 heavy frequencies 被这些 $t_j$ 捕获，translation invariance 就可以证明。若 $v$ 在 sketch kernel 中，即对所有生成元 $\langle t_j,v\rangle\in\mathbb{Z}$，那么对每个 heavy $\zeta$，$\langle \zeta,v\rangle$ 也近似为整数；非 heavy 频率在卷积 $\widehat{\nu}=\prod_i\widehat{\mu_i}$ 中被乘法快速压小。直接在高维把 Fourier $\ell_2$ 控制转成 total variation 会损失太大，所以作者沿 $v$ 的方向逐条直线分析：固定一条 affine line，研究 $\nu(x+tv)$ 的一维 Fourier 行为，再用 Cauchy-Schwarz 控制这一维上的 $\ell_1$ 差距，最后对所有平行线求和。这一步把“频率都不敏感”转成了 $d_{\mathrm{TV}}(\nu,\tau_v\nu)$ 小。

mollified route 的想法更干净。对 smooth problems，不必精确落在 $y$，而是落在 $y+Z$，其中 $Z\sim\gamma_R$。若 $f(y+Z)$ 通常接近 $f(y)$，这个额外噪声不影响目标答案；但 Fourier 侧多乘了一个 $\widehat{\gamma_R}(\zeta)\approx \exp(-\Theta(R^2\|\zeta\|_{\mathbb{T}^n}^2))$，远离原点的频率全部被指数压掉。因此只需要控制 near-origin large spectrum。near-origin 频率周期长，在半径 $R$ 的高斯尺度内只有常数多个有效周期；每加入一个近似独立的 near-origin heavy frequency，都会让高斯质量下降一个 $R^{\Theta(1)}$ 因子。密度上界 $2^{O(S)}$ 因而只允许
$$
O(S/\log R)
$$
个这样的方向。这就是 mollified theorem 维度少一个 $\log R$ 的原因，也是它能给出总空间 $O(S)$ 的原因。

结果的下界应用分两类。第一类通过 Gribelyuk, Lin, Woodruff, Yu, and Zhou (2025) 的 real-to-integer sketch lifting，把 real sketch dimension lower bounds 变成 bounded-entry integer sketch lower bounds，再由 Theorem 1.2 推到 polynomial-length turnstile streaming。论文给出例如 $\ell_p$ estimation：
$$
\textbf{Theorem 1.3.}\quad
\begin{cases}
1\le p\le 2:\quad \Omega\!\left(\min\{\epsilon^{-2}\log(1/\delta),n\}\cdot \log W\right),\\
p>2:\quad \Omega\!\left(\min\{n^{1-2/p}\epsilon^{-2/p}\log n\log^{2/p}(1/\delta)
+n^{1-2/p}\epsilon^{-2}\log(1/\delta),\ n\}\cdot \log W\right),
\end{cases}
$$
适用于长度至多 $W\ge \mathrm{poly}(n/\delta)$ 的 unit update streams。这里的 $\log W$ dependence 是重点：它把 sketch dimension lower bound 以 tight 的方式转成了 bit lower bound。

第二类通过 public-coin SMP lower bounds。linear sketch 天然给 SMP protocol：每个 player 对自己的向量发 $Ax^{(i)}$，referee 相加得到 $A(\sum_i x^{(i)})$ 再解码。因此已有的 SMP 下界可以经 exact transfer 推到 polynomial-length strict turnstile，甚至 ROBP。代表性结论包括：
$$
\textbf{Theorem 1.4.}\quad
\begin{aligned}
&L_0\text{ estimation: }\Omega\!\left(
\frac{\epsilon^{-2}\log n\log\log W}
{\log(\epsilon^{-2}\log n\log\log W)}
\right)\text{ bits},\\
&\text{且当 }W\ge(\epsilon^{-2}\log n)^{\Omega(\epsilon^{-2}\log n)}
\text{ 时提升到 }
\Omega(\epsilon^{-2}\log n\log\log W);\\
&\text{approximate maximum matching: }\Omega(n^{2-3\epsilon-o(1)})\text{ bits}.
\end{aligned}
$$

这篇的技术价值不只是把一个开放区间补上，而是把“streaming-to-sketching equivalence”从 transition graph 语言换成了 Fourier/additive-combinatorics 语言。旧方法解释的是状态图足够久以后会混合；新方法解释的是一个 $S$-bit 状态最多只能留下 $O(S)$ 个加性频率障碍。这个解释更贴近 sketch lower bound 的现代工具，也更清楚地说明 KP20 的短流分离为什么没有直接延伸到 polynomial length：分离需要持续利用 intermediate-state promises，而在足够长但仍 polynomial 的 noisy landing 实验里，非线性状态若不能对应到少数 Fourier 障碍，就会被卷积噪声抹平。

和本周前几篇 streaming 论文也能形成对照。[2604.20394](https://arxiv.org/abs/2604.20394) 是为具体统计任务 pin down 一遍扫描空间复杂度，[2604.21922](https://arxiv.org/abs/2604.21922) 是用 non-redundancy 刻画 CSP streaming decidability；本文则更像一条元定理，说明在 polynomial-length turnstile 世界里，许多“只会对 linear sketch 证明”的 lower bound 其实已经足够。最后 exact route 里残留的 $O(\log S)$ overhead 很具体：不是概念上不该 tight，而是 torus lattice rounding 的误差传播还没有完全压住。这个瓶颈比“能否等价”本身小得多，也更容易成为下一步技术优化的靶子。

## 未入选但值得关注

- **The Exact Replica Threshold for Nonlinear Moments of Quantum States** [arXiv:2604.22627](https://arxiv.org/abs/2604.22627)：论文刻画 quantum-state nonlinear moment estimation 的 exact replica threshold，结果陈述很清楚；今天的完整解读优先放在更偏 TCS streaming 下界基础设施的论文上。
- **Dynamic Planar Graph Isomorphism is in DynFO** [arXiv:2604.22365](https://arxiv.org/abs/2604.22365)：把 dynamic planar graph isomorphism 放进 DynFO，说明一阶动态维护模型在平面图同构上仍有表达力；目前摘要层面更像 LICS full version 的模型推进，技术细节适合后续专题再展开。
- **Counting All Lattice Rectangles in the Square Grid in Near-Linear Time** [arXiv:2604.22456](https://arxiv.org/abs/2604.22456)：给出 square grid 中 lattice rectangles 的 near-linear exact counting，并配有二项渐近式；问题相对专门，所以今天没有放进主文。
- **Cuts and Gauges for Submodular Width** [arXiv:2604.22663](https://arxiv.org/abs/2604.22663)：用 cuts 和 gauges 重述 submodular width，并讨论它与 $\mathrm{ghw}/\log \mathrm{ghw}$ 的关系；结构上值得关注，但更偏数据库查询宽度理论，暂列备选。
